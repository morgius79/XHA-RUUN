# Глава 24: Спутники Théxar — Serrin и Thel

> *«Две луны — два глаза ночи. Serrin — мудрый, полный, смотрящий вперёд. Thel — быстрый, дикий, смотрящий в прошлое. Вместе они освещают нашу тьму.»*
> — Из «Песни двух лун», народная поэзия Xha'Ruun

---

**Связанные разделы:** → см. [Гл.03, «Планета Théxar»], → см. [Гл.16, «Магнитосфера»]
**Ключевой принцип:** Théxar имеет два спутника: Serrin (крупный, 425 км) и Thel (захваченный астероид, 178 км). Serrin стабилизирует ось планеты; Thel — источник He-3.

---

| Параметр | Значение |
|----------|----------|
| Радиус | 425 км |
| Масса | 3.8 × 10²⁰ кг |
| Орбитальный период | 31.4 сут |
| Эксцентриситет | 0.028 |
| Наклон к экватору | 4.2° |
| Состав | Силикатный (базальт, анортозит) |
| Атмосфера | Отсутствует |
| Гравитация | 0.14 м/с² |

## 24.2 Thel

| Параметр | Значение |
|----------|----------|
| Радиус | 178 км |
| Масса | 0.9 × 10²⁰ кг |
| Орбитальный период | 78.2 сут |
| Эксцентриситет | 0.18 |

## 24.3 Приливы

Совместное влияние Serrin и Thel создаёт сложный приливной режим.
