## 27.1 Геодезическая система

### 27.1.1 Референц-эллипсоид

Система координат Théxar базируется на референц-эллипсоиде **WGS-Théxar**, утверждённом Гильдией Навигаторов в +2 200 г. Э.Е.

| Параметр | Значение |
|----------|----------|
| Экваториальный радиус (a) | 1 242 мо |
| Полярный радиус (b) | 1 238 мо |
| Сжатие (f) | 1/298.3 |
| Средний радиус (R₀) | 1 241 мо |
| Окружность экватора | 7 805 мо |
| Окружность меридиана | 7 781 мо |
| Площадь поверхности (эллипсоид) | 19.4 млн мо² |
| Объём | 8.03 × 10⁹ мо³ |

### 27.1.2 Геоид

Геоид Théxar — модель формы планеты, учитывающая гравитационные аномалии. Отклонения геоида от WGS-Théxar:

| Регион | Аномалия (ша) | Причина |
|--------|-------------|---------|
| Хребет Khar-Tor | +51 ша | Гравитационный избыток (горная масса) |
| Впадина Thash | −46 ша | Гравитационный дефицит (глубоководная впадина) |
| Плато Raal | +22 ша | Избыточная плотность коры |
| Пустыня Shan | −15 ша | Тонкая кора |

## 27.2 Система координат

### 27.2.1 Географические координаты

Xha'Ruun используют сферическую систему координат:

- **Широта (φ):** 0° (экватор) → +90° (север) / −90° (юг)
- **Долгота (λ):** 0° (нулевой меридиан) → +180° (восток) / −180° (запад)
- **Высота (h):** над референц-эллипсоидом WGS-Théxar

### 27.2.2 Нулевой меридиан

Нулевой меридиан проходит через **центр площади Правительственного квартала Khar-Vexar**, точку с координатами:

- WGS-Théxar: φ = 42°18'24" с.ш., λ = 0°00'00"
- Отметка: бронзовая пластина диаметром 1.2 ша в центре площади

### 27.2.3 Экватор

Экватор Théxar пересекает южную оконечность континента Kharak и северную — Rhuval. В точке пересечения с меридианом 48° в.д. установлен **Памятник Экватору** — 15-шаговая стела из чёрного базальта.
