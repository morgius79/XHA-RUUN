# Приложение 2: Глоссарий терминов

Полный словарь терминов энциклопедии Xha'Ruun. Формат: **термин** [язык] — определение. → см. соответствующую главу для контекста.

---

**Khar'Vex** [Xha'Ruun] — звезда системы, оранжевый карлик K2V. «Подательница света». → см. [Гл.02]

**Kharak** [Xha'Ruun] — крупнейший континент Théxar (34.2 млн км²). «Земля света». → см. [Гл.03]

**Rhuval** [Xha'Ruun] — второй по величине континент Théxar (21.7 млн км²). «Великая земля». → см. [Гл.03]

**Vexar** [Xha'Ruun] — третий континент Théxar (18.5 млн км²). «Творящая земля». → см. [Гл.03]

**Sha'Ruun** [Xha'Ruun] — язык Xha'Ruun. «Язык народа». → см. [Гл.06]

**Théxar** [Xha'Ruun] — планета, родина Xha'Ruun. «Земля-мать». → см. [Гл.03]

**Serrin** [Xha'Ruun] — крупнейший спутник Théxar (R=425 км). → см. [Гл.03]

**Thel** [Xha'Ruun] — малый спутник Théxar (R=178 км). → см. [Гл.03]

**Thalass** [Xha'Ruun] — крупнейший спутник Rhuun (R=2 840 км). Кандидат на колонизацию. → см. [Гл.02]

**Xha'Ruun** [Xha'Ruun] — разумный вид. «Мыслящий народ» (полная форма: Xha'Ruun-Vokh). → см. [Гл.05]

## А

**Агглютинация** [лингв.] — способ словообразования, при котором к корню последовательно присоединяются суффиксы, каждый с одним значением. Характерна для Sha'Ruun. → см. [Гл.06]

**Аммиачная модификация** [биохим.] — присутствие ~3% NH₃ во внутриклеточной жидкости организмов Théxar. Криопротектор, модификатор растворимости. → см. [Гл.04]

**Астеносфера** [геол.] — частично расплавленный слой верхней мантии Théxar (глубина 410–660 км). → см. [Гл.03]

---

## В

**Вексы** (vex-gran) [Xha'Ruun] — верхняя пара конечностей Xha'Ruun, эволюционно бывшие крыльями. Манипуляторы + рудиментарное планирование. → см. [Гл.05]

**Вексаний (Vx)** [хим.] — сверхтяжёлый элемент (Z=128, A=330). Плотность 24 г/см³. Применение: балласт, радиационная защита. → см. [Гл.09]

**Великое Дыхание** (Mor-Vokh) [религ.] — цикл расширения и сжатия вселенной. → см. [Гл.10]

**Вен-Тхар (VT)** [экон.] — валюта Xha'Ruun. 1 VT = 1 кВт·ч термоядерной энергии. → см. [Гл.08]

**Вибрация Первичная** (Khar-Vokh) [религ.] — источник всего сущего, аналог квантовой флуктуации. → см. [Гл.10]

**Гемоцианин** [биох.] — медьсодержащий пигмент крови Xha'Ruun, придающий ей сине-зелёный цвет. → см. [Гл.05]

**Гильдии** [соц.] — 12 профессиональных объединений Xha'Ruun. → см. [Гл.08]

**Голоса** (Семь Голосов) [религ.] — семь первичных сущностей, «спевших» вселенную. → см. [Гл.10]

**Гхарра** [геогр.] — Южный океан Théxar (87.5 млн км²). → см. [Гл.03]

## Д

**Долг знания** (ven-mor) [этика] — моральное обязательство делиться знанием с архивами. → см. [Гл.08]

**Инфразвуковая чувствительность** [биол.] — способность Xha'Ruun слышать 8–20 Гц. Наследие предков для обнаружения землетрясений. → см. [Гл.05]

## К

**Календарь Xha'Ruun** — 14 месяцев по ~29.5 суток + 5 дней Обновления. Текущий год: 4 028 Э.Е. → см. [Гл.08]

**Квантрофон** [тех.] — музыкальный инструмент, создающий звук на квантовых флуктуациях в сверхпроводнике. → см. [Гл.08]

**Квартетный слух** [биол.] — способность определять направление звука с точностью ±0.5° благодаря 4 ушам. → см. [Гл.05]

**Красная гниль** [ист.] — эпидемия −2 120 – −2 105 гг. цив. Убила ∼200 млн Xha'Ruun. → см. [Гл.07]

**Кхарний (Kh)** [хим.] — сверхтяжёлый элемент (Z=127, A=327). Ферромагнетик, сверхпроводник. Период полураспада: 12 400 лет. → см. [Гл.09]

**Кхарру** [биол.] — одомашненное травоядное (аналог КРС). Шестиногое, класс Thermochromadae. → см. [Гл.05]
