## Граф порядка написания

```
01 ──→ 02 ──→ 03 ──→ 04 ──→ 05 ──→ 06 ──→ 07 ──→ 08 ──→ 09
                                         ↓           ↓
                                          └────→ 10 ←─┘
```

Строго последовательный: каждая глава опирается на все предыдущие.

---

## Словарные термины для MAP.md

| Термин | Определение |
|--------|-------------|
| Зависит от | Глава использует факты, установленные в указанных главах |
| Влияет на | Глава поставляет факты для указанных глав |
| Конфликт | Ситуация, когда две главы устанавливают противоречивые факты |
| Ревизия | Изменение CANON.md, требующее проверки всех зависимых глав |
