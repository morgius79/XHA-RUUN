## 8.1 Root System

The Xha'Ruun language is based on ~500 roots. Each root is monosyllabic (CVC or CV).

| Root | Meaning | Derivatives |
|------|---------|-------------|
| **xha-** | thought, mind | xha'ruun (people), xha'ven (truth) |
| **khar-** | light, star | khar'vex (luminary), kharan (light) |
| **the-** | soil | thexar (planet), theri (ground) |
| **sha-** | tongue, word | sharuu (writing), shaven (dialect) |
| **ruun-** | people | ruuna (community), ruunen (society) |

## 8.2 Word Formation

| Method | Suffix | Example |
|--------|--------|---------|
| Noun | -∅ | **khal** (life) |
| Adjective | -an | **kharan** (light) |
| Verb | -en | **khalen** (to live) |
| Abstraction | -ven | **khalven** (vitality) |
| Agent | -sha | **khalsha** (living one) |
| Place | -the | **khalthe** (habitat) |
