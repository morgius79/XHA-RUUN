# Энциклопедия цивилизации Xha'Ruun

Научно-фантастическая энциклопедия языка и цивилизации Xha'Ruun (500–600 страниц).

**Стиль:** академический атлас (National Geographic / DK Publishing)
**Язык:** русский
**Формат:** Markdown

## Структура

```
xharuun/
├── CANON.md                 # Центральный файл согласованности (все факты)
├── MAP.md                   # Матрица зависимостей глав
├── README.md                # Этот файл
│
├── templates/               # Шаблоны
│   ├── chapter-template.md

├── 05-species-xharuun/      # Разумный вид Xha'Ruun
├── 06-language/             # Язык и письменность
├── 07-history/              # История цивилизации
├── 08-culture/              # Культура и общество

```

## Правила работы

1. **Согласованность превыше всего.** Любой новый факт проверяется на совместимость с CANON.md.
2. **Главы пишутся последовательно.** Фундаментальные главы (01–03) задают рамки для последующих.
3. **Одна глава — одна директория.** Основное содержание в `index.md`.
4. **Кросс-ссылки обязательны.** Формат: `→ см. [Гл.NN, "Раздел"]`.
