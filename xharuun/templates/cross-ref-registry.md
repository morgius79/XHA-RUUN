# Реестр кросс-ссылок энциклопедии Xha'Ruun

## Формат ссылок

В тексте используются ссылки единого формата:

```
→ см. [ТИП:ID, "Контекст"]
```

Где ТИП — один из:

| Тип | Значение | Пример |
|-----|----------|--------|
| Гл.NN | Глава (01–10) | → см. [Гл.03, "Геология"] |
| ТАБ:ID | Таблица | → см. [ТАБ:planetary-data] |
| ПРИЛ:ID | Приложение | → см. [ПРИЛ:timeline] |
| СЛ:ID | Словарный термин | → см. [СЛ:khar-vex] |
