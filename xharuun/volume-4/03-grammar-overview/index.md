## 3.1 Типологические характеристики

| Параметр | Значение |
|----------|----------|
| Тип | Агглютинативная, суффиксальная |
| Порядок слов | SOV (субъект-объект-глагол) |
| Падежи | 14 |
| Числа | 3 (единственное, двойственное, множественное) |
| Род | Отсутствует |
| Эвиденциальность | Обязательная (4 типа) |
| Время | 3 + 1 (прошлое, наст., будущее + вневременное) |

## 3.2 Структура слова

```
Основа-(корень) + суффикс_числа + суффикс_падежа + (эвиденциальность)
```

Пример: **xha'ruun-ak-sha-ven** → «народ-МН-ЭРГ-ПРЯМ» = «народы (я вижу, что они)»
