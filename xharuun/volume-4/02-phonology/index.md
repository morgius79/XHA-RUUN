## 2.1 Фонотактика

Структура слога: (C)(C)V(C)(C)

| Позиция | Возможные звуки |
|---------|----------------|
| C₁ (инициаль) | Любой согласный |
| C₂ (инициаль) | r, l, w, j |
| V (вершина) | Любой гласный |
| C₃ (финаль) | Любой согласный |
| C₄ (финаль) | s, ʃ, t, d, n |

## 2.2 Ударение

Динамическое, на первый слог корня. Нефонематическое (не меняет значение слова).

## 2.3 Гармония гласных

Ассимиляция по ряду: суффиксы подстраиваются под гласную корня.
