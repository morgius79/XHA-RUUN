📡 Keitaro — установка и настройка

Трекер для баинга: VPS, установка, офферы, потоки, postback, связка с Meta

Meta Ads → Keitaro → Преленд → Оффер

1. Что такое Keitaro и зачем он нужен

Meta (креатив) → Keitaro → Преленд → Оффер

⬇ Сбор статистики: клики, конверсии, ГЕО, устройство, источник

Зачем:

  1. Маскировка ссылки — Meta не видит, куда ведёшь
  2. Распределение трафика — разные офферы под разные условия
  3. Статистика — какой креатив/канал/ГЕО дают конверсии
  4. Anti-cheat — отсекает ботов
  5. Postback — передаёт конверсии обратно в Meta

2. Требования к серверу

ПараметрМинимумРекомендуется

Где брать: DigitalOcean ($6/мес), Vultr ($6/мес), Hetzner (от $4/мес). IP не должен быть забанен в Meta.

3. Установка

Быстрая (рекомендуется)

ssh root@твой-сервер
cd /opt
wget https://keitaro.io/download/keitaro-latest.zip
unzip keitaro-latest.zip -d keitaro
cd keitaro
chmod +x install.sh
./install.sh

Ручная (если авто не сработал)

apt update
apt install -y nginx mysql-server php8.2-fpm php8.2-mysql php8.2-curl \
  php8.2-xml php8.2-mbstring php8.2-zip php8.2-gd php8.2-bcmath unzip wget

cd /var/www
wget https://keitaro.io/download/keitaro-latest.zip
unzip keitaro-latest.zip
mv keitaro tracker
chown -R www-data:www-data tracker

# Создать БД
mysql -u root -p
CREATE DATABASE keitaro;
CREATE USER 'keitaro_user'@'localhost' IDENTIFIED BY 'твой_пароль';
GRANT ALL PRIVILEGES ON keitaro.* TO 'keitaro_user'@'localhost';
FLUSH PRIVILEGES;
EXIT;

# Настроить Nginx (см. полный конфиг в md-файле)

После установки

  • Открой http://tracker.mydomain.com/admin
  • Смени пароль админа
  • Настрой часовой пояс под ГЕО
  • Закрой панель от чужих — добавь HTTP-авторизацию

4. Основные понятия

OFFER (Оффер) — ссылка на ленд партнёрки
STREAM (Поток) — правила распределения трафика
CAMPAIGN (Кампания) — твоя кампания в Meta (ссылка из Keitaro)
STATS (Статистика) — клики, конверсии, доход, расход

5. Настройка оффера

Offers → Add Offer

ПолеЧто вводить
NameCasino ID (внутреннее название)
Offer URLhttps://partner.casino.com/click?subid={clickid}&campaign=123
PayoutВыплата партнёрки
CountryГЕО оффера

Важно: {clickid} — ID клика в Keitaro. Партнёрка вернёт его при конверсии.

8. Postback — самое важное

Meta → Keitaro → Оффер → депозит
Партнёрка → callback с clickid → Keitaro → конверсия в Meta

Настройка postback в Keitaro:

  1. Offers → выбери оффер → Postback
  2. Добавь URL от партнёрки: https://partner.casino.com/postback?clickid={clickid}&payout={payout}
  3. Параметры: Click ID = {clickid}, Payout = {payout}

Чтобы конверсии доходили до Meta:

СпособКак настроитьНадёжность
FB PixelКод пикселя на преленде🟡 Средняя
CAPI ServerPixel ID + Access Token в Keitaro🟢 Высокая

9. Полная схема Meta → Keitaro → Оффер

1. Создаёшь поток в Keitaro → получаешь ссылку click/XXXXX
2. Вставляешь ссылку в объявление Meta
3. Настраиваешь postback (партнёрка → Keitaro → Meta)

Как работает:
Meta показывает креатив → Петя кликает → Meta записывает click_id
→ Петя в Keitaro → click_id сохраняется → преленд → оффер
→ Петя регится и делает депозит
→ Партнёрка шлёт callback в Keitaro
→ Keitaro шлёт конверсию в Meta
→ Meta оптимизирует показы под таких как Петя

13. Чек-лист настройки

Установка

  • VPS арендован (2+ GB RAM)
  • Домен привязан к VPS
  • Keitaro установлен
  • SSL-сертификат (HTTPS)
  • Админ-панель закрыта паролем

Офферы и потоки

  • Оффер создан, Offer URL содержит {clickid}
  • Выплата указана
  • Поток создан, преленд добавлен
  • Ссылка потока скопирована

Meta и postback

  • Ссылка из Keitaro вставлена в объявление
  • FB Pixel стоит на преленде (или CAPI)
  • Postback URL от партнёрки добавлен
  • Тестовая конверсия прошла
  • Конверсии видны и в Keitaro, и в Meta

14. Безопасность

  • Закрой admin-панель — HTTP-авторизация или по IP
  • Обновляй Keitaro — новые версии закрывают уязвимости
  • Поддомены: панель на одном, трек-ссылки на другом
  • Бэкапы: дамп БД раз в неделю
mysqldump -u root keitaro > /backup/keitaro_$(date +%Y%m%d).sql
