# L-000451: xha-ruuni

**Корень:** нет
**Часть речи:** существительное
**Перевод:** столица, главный город
**Статус:** active
**Создан:** 2026-07-16

## Этимология

xha- (разум) + ruun- (народ) + -i (место): центр разума народа

## Примеры

1. Xha-ruuni thel ruuni-eth ran xha-ruun. — Столица — поселение всего народа.
2. Xha-ruuni veni thero. — В столице есть архив.
