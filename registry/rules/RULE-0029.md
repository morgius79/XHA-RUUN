# RULE-0029: Перекрёстные ссылки в реестрах

**Тип:** канон
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Перекрёстные ссылки между объектами канона оформляются в виде маркированного списка в секции «Связи» карточки объекта, с указанием ID и краткого описания отношения.

## Описание

Ссылки записываются в формате «- [ID]: описание связи». Допускаются ссылки на объекты других категорий. При изменении целевого объекта (retired, перенос) ссылка должна быть проверена и при необходимости обновлена. Ссылки на retired объекты допускаются с пометкой «(retired)» после ID. Ссылка на несуществующий ID считается нарушением целостности.

## Примеры

1. Соблюдение: «- L-000145: корень для RULE-0005»; «- CH-0003: персонаж, впервые упомянутый в LOC-0008».
2. Нарушение: ссылка на ID без указания категории; ссылка на несуществующий ID; отсутствие пометки retired при ссылке на удалённый объект.
