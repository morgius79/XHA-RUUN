# RULE-0050: Автоматизация сборки (CI/CD)

**Тип:** технические
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Сборка, валидация и публикация канона Xha'Ruun автоматизированы через скрипты в директории build/ и (опционально) CI-пайплайн GitHub Actions.

## Описание

Конвейер включает этапы: (1) validate — проверка всех правил (RULE-0047); (2) check-links — проверка ссылок (RULE-0049); (3) build-pdf — сборка PDF (RULE-0046); (4) build-font — сборка шрифта (RULE-0048); (5) release — создание тега и публикация артефактов. Основной скрипт: build/run-all.bat (.ps1). CI-пайплайн (если настроен) запускается при пуше в main и создании тега v*.

## Примеры

1. Соблюдение: после пуша в main GitHub Actions запускает validate — check-links — build-pdf и прикрепляет PDF к релизу.
2. Нарушение: публикация релиза без прогона валидации; пропуск этапа проверки ссылок; ручная сборка PDF без запуска конвейера.
