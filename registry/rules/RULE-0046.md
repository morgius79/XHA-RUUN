# RULE-0046: Сборка PDF из Markdown

**Тип:** технические
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

PDF-версия канона собирается из всех markdown-файлов директорий encyclopedia/ и registry/ с помощью Pandoc, используя единый шаблон-темплейт в формате LaTeX.

## Описание

Команда сборки: pandoc encyclopedia/*.md registry/**/*.md -o xharuun-canon.pdf --template=xharuun.tex. Шаблон включает: титульную страницу, оглавление, колонтитулы с версией и датой. Порядок глав определяется файлом _toc.md. Карточки реестра включаются в приложение. Шрифт для языка Xha'Ruun подключается отдельно (см. RULE-0048). Сборка выполняется скриптом build/build-pdf.bat.

## Примеры

1. Соблюдение: запуск build-pdf.bat создаёт xharuun-canon.pdf с корректными оглавлением и шрифтами.
2. Нарушение: пропуск файлов реестра в сборке; отсутствие шаблона; сборка без указания версии на титульной странице.
