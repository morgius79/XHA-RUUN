# RULE-0024: Патронимическая конструкция

**Тип:** именование
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Патроним образуется добавлением родительного падежа имени родителя (-ash/-esh) после личного имени через пробел: [имя] [родитель-ash].

## Описание

Патроним указывает на происхождение и используется для формальной идентификации. В культуре Xha'Ruun используется патроним по отцу или родителю нейтер (по тому из родителей, чей статус выше). Имя родителя ставится в генетиве (родительном падеже). Патроним не склоняется; склоняется только личное имя. При представлении порядок: имя, патроним, возможный топоним.

## Примеры

1. Соблюдение: «Muri-Xhaena Khal-Ruunen-ash» (Muri-Xhaena, дочь Khal-Ruunen), «Zer-Tharun Murina-ash» (Zer-Tharun, сын Murina).
2. Нарушение: патроним без падежного окончания; патроним в номинативе; патроним, записанный через дефис с именем.
