# RULE-0025: Транслитерация (латинизация)

**Тип:** именование
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Для латинизации Xha'Ruun используется строгий набор из 22 латинских букв без диакритики: a, b, d, e, g, h, i, k, l, m, n, o, p, r, s, t, u, v, x, y, z, sh (диграф).

## Описание

Апостроф (') обозначает гортанную смычку или раздельное произнесение гласных на стыке основ. Диграф «sh» соответствует /ʃ/. Буквы c, f, j, q, w не используются в стандартной орфографии; они допустимы только в заимствованиях и мета-комментариях, где выделяются курсивом. Удвоение гласных обозначает долготу (aa, ii, uu). Удвоение согласных обозначает геминацию.

## Примеры

1. Соблюдение: «Xha'Ruun» (апостроф для раздельного произнесения Xha + Ruun), «khalash» (допустимый диграф sh), «tharu» (th — допустимое сочетание t+h).
2. Нарушение: использование буквы c в Xha'Ruun-слове; отсутствие апострофа там, где две гласные должны произноситься раздельно; использование диакритики (é, ü, ñ).
