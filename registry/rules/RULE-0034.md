# RULE-0034: Структура директорий реестра

**Тип:** канон
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Директория registry/ содержит поддиректории graphemes/, lexemes/, rules/, locations/, events/, characters/; каждая поддиректория содержит файлы карточек и опциональную поддиректорию _retired/.

## Описание

Структура: registry/{category}/{ID}.md, registry/{category}/_index.md, registry/{category}/_retired/{ID}.md. Файл _index.md содержит перечень всех зарегистрированных (включая retired) объектов категории с ID, названием, типом и статусом. Файлы карточек следуют единому шаблону для категории. Поддиректория _retired содержит карточки удалённых объектов.

## Примеры

1. Соблюдение: registry/rules/RULE-0001.md, registry/rules/_index.md, registry/rules/_retired/ (если есть retired).
2. Нарушение: карточка объекта не в своей категории; отсутствие _index.md; файл карточки в корне registry/.
