# RULE-0039: Формат карточек реестра

**Тип:** стиль
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Все карточки реестров следуют единому шаблону с секциями: заголовок H1 с ID и названием, метаданные (Тип, Статус, Создан), Формулировка, Описание, Примеры, Связанные правила.

## Описание

Шаблон унифицирован для всех категорий. Метаданные оформляются полужирным шрифтом в формате «**Поле:** значение». Секции отделяются пустой строкой. Примеры нумеруются арабскими цифрами с точкой. В секции «Связанные правила» указываются только прямые зависимости, не все возможные ссылки. Для графем и лексем добавляется секция «Фонетика» или «Этимология».

## Примеры

1. Соблюдение: карточка RULE-0001 следует шаблону с пятью секциями; карточка L-000001 имеет дополнительную секцию «Этимология».
2. Нарушение: произвольный порядок секций; отсутствие обязательной секции; метаданные не в формате «**Поле:** значение».
