# RULE-0019: Нейтральные окончания (нейтер)

**Тип:** именование
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Существительные и имена среднего рода (нейтер) оканчиваются на -in (краткое) или -el (долгое).

## Описание

Нейтер используется для неодушевлённых объектов, абстрактных понятий, божественных сущностей без пола и лиц неопределённого или третьего гендера. Окончание -in — стандартное, -el — для сакральных и канонических терминов. В именах нейтер может использоваться для персонажей, чья сущность выходит за рамки бинарной гендерной системы — пророков, бесполых существ, искусственных интеллектов.

## Примеры

1. Соблюдение: «Xha'Ruunin» (канонический термин «Путь Закона»), «Tharel» (сакральный посланник), «Khalin» (жилище как абстракция).
2. Нарушение: окончание -in для одушевлённого персонажа с известным полом; окончание -el для бытового предмета; смешение нейтер с женским или мужским согласованием.
