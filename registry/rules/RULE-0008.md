# RULE-0008: Временна́я система глагола

**Тип:** грамматика
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Глагол Xha'Ruun различает четыре времени: настоящее, прошедшее, будущее и вневременное (гномическое), маркируемые суффиксами после аспекта.

## Описание

Временные суффиксы присоединяются к аспектуальной основе глагола. Настоящее время (-Ø нулевой) обозначает действие в момент речи. Прошедшее (-ar/-er) — действие до момента речи. Будущее (-un/-yn) — действие после момента речи. Вневременное (-ash/-esh) используется для общих истин, определений, законов природы и канонических утверждений. Вневременное время обязательно в текстах канона.

## Примеры

1. Соблюдение: «xharun-Ø-ku» (бежать-PRS-1SG — «я бегу»); «xharun-ar-ku» (бежать-PST-1SG); «xharun-un-ku» (бежать-FUT-1SG); «xharun-ash» (бежать-GNO — «бег существует / бег есть» — каноническая форма).
2. Нарушение: использование настоящего времени для канонических утверждений; смешение показателей времени и аспекта.
