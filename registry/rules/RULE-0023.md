# RULE-0023: Титулы и гоноративы

**Тип:** именование
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Титулы и гоноративы предшествуют имени в виде отдельных слов и пишутся с заглавной буквы; после титула ставится пробел, затем полное двухосновное имя.

## Описание

Система титулов включает: Tharu (Хранитель/Страж), Xhaesi (Учитель/Наставник канона), Khalen (Старейшина), Zar (Владыка/Правитель), Rhu (Судья/Толкователь). Титулы не склоняются. При обращении допустимо использование сокращённого титула без имени (Tharu = «Страж»). Титул не заменяет имени, а предшествует ему. В официальных текстах канона титул обязателен при первом упоминании персонажа.

## Примеры

1. Соблюдение: «Tharu Khal-Ruunen» (Страж Khal-Ruunen), «Xhaesi Muri-Xhaena» (Учитель Muri-Xhaena), «Zar Zer-Tharun» (Владыка Zer-Tharun).
2. Нарушение: титул после имени («Khal-Ruunen Tharu»); склонение титула; множественный титул без конъюнкции.
