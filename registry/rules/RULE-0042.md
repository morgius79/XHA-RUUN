# RULE-0042: Формат языковых примеров

**Тип:** стиль
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Языковые примеры в текстах канона оформляются в виде маркированного списка с курсивом для фразы на Xha'Ruun и переводом в скобках.

## Описание

Формат: «*[фраза на Xha'Ruun]* (морфемная глосса) — «перевод на русский».» Морфемная глосса даётся через дефис для каждой морфемы. В правилах грамматики примеры нумеруются. В энциклопедии примеры даются внутри абзаца курсивом с переводом в кавычках. Длинные примеры (более 5 слов) выносятся в отдельный блок.

## Примеры

1. Соблюдение: «*khal-u-nar* (дом-PL-DAT) — «домам»»; «*Xha'Ruun khal-ar tharun-ar-thu* — «Посланник Xha'Ruun прибыл домой».
2. Нарушение: пример без перевода; отсутствие курсива для Xha'Ruun-текста; морфемная глосса без дефисов.
