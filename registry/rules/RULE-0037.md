# RULE-0037: Оформление таблиц

**Тип:** стиль
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Таблицы оформляются в стандартном Markdown-формате GFM с заголовками, выравниванием через дефисы, и без ячеек с пустым содержимым (использовать тире «—» для отсутствующих данных).

## Описание

Таблица начинается с строки заголовков, затем строка выравнивания (--- для обычного, :--- для левого, ---: для правого, :---: для центрированного), затем строки данных. Пустые ячейки не допускаются: отсутствие данных обозначается тире «—». Длинные таблицы (более 15 строк) сопровождаются кратким примечанием-подписью после таблицы курсивом.

## Примеры

1. Соблюдение: «| Падеж | Суффикс | Пример |\n|-------|---------|--------|\n| Номинатив | -Ø | khal |\n| Аккузатив | -u | khal-u |».
2. Нарушение: пустая ячейка; отсутствие строки выравнивания; таблица без заголовков; использование HTML-таблиц вместо Markdown.
