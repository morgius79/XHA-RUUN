# RULE-0045: YAML-фронтматер документов

**Тип:** стиль
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Документы проекта, за исключением карточек реестра, могут содержать YAML-фронтматер с полями: title, description, version, date, status.

## Описание

Фронтматер заключается между тройными дефисами (---) в начале файла. Поля: title (строка), description (строка), version (SemVer), date (YYYY-MM-DD), status (draft | review | published | archived). Для карточек реестра фронтматер не используется, так как метаданные встраиваются в тело карточки. Фронтматер обязателен для файлов ROADMAP.md, RELEASE.md и PROJECT_CONTEXT.md.

## Примеры

1. Соблюдение: «---\ntitle: ROADMAP\ndescription: План развития проекта Xha'Ruun\nversion: 0.1.1\ndate: 2026-07-16\nstatus: published\n---».
2. Нарушение: фронтматер в карточке реестра; отсутствие обязательного поля; неверный формат даты; статус не из разрешённого набора.
