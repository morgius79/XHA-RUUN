# RULE-0043: Аннотация карт и диаграмм

**Тип:** стиль
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Карты, диаграммы и схемы сопровождаются подписью в формате «Рис. N — Описание» и легендой, если содержат условные обозначения.

## Описание

Изображения вставляются через стандартный Markdown-синтаксис: «![Подпись](путь)». Подпись начинается с «Рис. N — », где N — номер рисунка в пределах главы. Если изображение содержит условные обозначения (цвета, символы), обязательна легенда после рисунка в виде маркированного списка. Карты населённых пунктов должны включать ссылки на соответствующие LOC-ID.

## Примеры

1. Соблюдение: «![Рис. 1 — Карта региона Khalkashath](images/map-khalkashath.png)» с легендой «— столица (LOC-0001)».
2. Нарушение: рисунок без подписи; отсутствие легенды при условных обозначениях; номер рисунка не соответствует порядку в главе.
