# RULE-0038: Синтаксис перекрёстных ссылок

**Тип:** стиль
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Перекрёстные ссылки в тексте канона оформляются как [ID] (в квадратных скобках) при первом упоминании объекта; при повторном — как ID в круглых скобках или контекстное упоминание.

## Описание

При первом упоминании объекта в главе используется формат «[ID]: Название» (например, «[LOC-0001]: Khalkashath — столица...»). При повторных упоминаниях — краткая форма «(LOC-0001)». Ссылки на правила: «(согласно RULE-0014)». Ссылки должны вести на существующие ID; битые ссылки выявляются при валидации.

## Примеры

1. Соблюдение: «Столица цивилизации — [LOC-0001]: Khalkashath. Жители Khalkashath (LOC-0001) чтят Xha'Ruun (RULE-0014).»
2. Нарушение: ссылка на несуществующий ID; формат [ID] при повторном упоминании; ссылка без квадратных скобок при первом упоминании.
