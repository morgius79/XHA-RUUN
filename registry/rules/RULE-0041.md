# RULE-0041: Цитирование и атрибуция источников

**Тип:** стиль
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Цитаты из канонических источников оформляются в виде блоков с указанием источника в скобках после цитаты: «> [текст] (ID источника)».

## Описание

Цитаты из энциклопедии, реестров или аудио-материалов выделяются блоком (>) с последующей атрибуцией в круглых скобках. Атрибуция включает ID источника и, при необходимости, номер главы или строки. Для цитат из баллад Suno добавляется метка времени. В мета-текстах (ROADMAP, RELEASE) цитаты оформляются без блоков, в кавычках.

## Примеры

1. Соблюдение: «> Порядок рождается из хаоса, как свет из тьмы (Энциклопедия Xha'Ruun, гл. 1)»; «> Kхаl u tharun (L-000012)».
2. Нарушение: цитата без указания источника; цитата внутри кавычек вместо блока; атрибуция без ID при наличии ID у источника.
