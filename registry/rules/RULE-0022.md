# RULE-0022: Названия событий

**Тип:** именование
**Статус:** active
**Создан:** 2026-07-16

## Формулировка

Исторические события именуются по формуле [основа_сути]-[основа_времени] с суффиксом -vash для событий внутри канона и -vor для событий мета-канона.

## Описание

События внутри вселенной Xha'Ruun получают суффикс -vash. Реальные события по созданию проекта — -vor. Первая основа отражает суть события (Mur- «свет/начало», Kash- «тьма/кризис», Thar- «прибытие/переход», Khal- «основание»). Вторая основа — время (ar «прошлое», en «настоящее», un «будущее»). Названия событий всегда пишутся с заглавной буквы.

## Примеры

1. Соблюдение: «Mur-Aravash» (Свет Прошлого — каноническое событие), «Khal-Unvor» (Основание Будущего — мета-событие, запуск проекта).
2. Нарушение: событие мета-канона с суффиксом -vash; бытовое происшествие с суффиксом имени события; отсутствие суффикса -vash/-vor.
