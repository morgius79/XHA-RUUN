# G-0061: Разделитель Слов

**Тип:** пунктуация
**Чтение:** (разделитель)
**Статус:** active
**Создан:** 2026-07-16

## Описание

Небольшая вертикальная точка, расположенная чуть ниже средней линии строки. Самый частотный пунктуационный знак письменности Шаруу — используется для отделения всех лексических слов друг от друга. В отличие от земных письменностей, пробелы между графемами внутри слова не ставятся: разделитель ставится только на границах слов.
