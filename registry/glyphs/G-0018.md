# G-0018 — KRA

**Транслитерация:** kra  
**Произношение:** /kra/  
**Тип:** Слоговой (CCV) — K-ряд, кластер  
**Слог:** kra

**Описание:** Кластерный знак K-ряда. Обозначает комплекс kra — один из допустимых трёхфонемных слогов.

**Употребление:**
- *kral* (твёрдый, прочный)
- *kra-thor* (основание, фундамент)

**Графическое строение (16×16):**
- K-основа (вертикаль + петля)
- R-инфикс: диагональный штрих
- A-маркер: две параллельные линии

**Связи:**
- ← G-0013 (ka) + R-инфикс
- → G-0019 (kla) — латеральный вариант
