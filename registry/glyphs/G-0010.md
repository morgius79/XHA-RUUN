# G-0010 — Ə

**Транслитерация:** ə  
**Произношение:** /ə/  
**Тип:** Гласный (V), нейтральный  
**Слог:** schwa

**Описание:** Нейтральный гласный (шва). Самый частотный гласный в безударных позициях. Выполняет роль грамматического «наполнителя».

**Употребление:**
- Грамматический аффикс по умолчанию
- Эпентеза (вставной гласный) в сложных кластерах
- Показатель неопределённости

**Графическое строение (16×16):**
- Три точки, расположенные треугольником
- Минимальный знак системы

**Связи:**
- Используется в грамматических шаблонах
- Один из самых частотных знаков
