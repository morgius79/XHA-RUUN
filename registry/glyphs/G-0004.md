# G-0004 — I

**Транслитерация:** i  
**Произношение:** /i/  
**Тип:** Гласный (V)  
**Слог:** i

**Описание:** Верхний гласный переднего ряда. Один из двух «малых» гласных наряду с u.

**Употребление:**
- Показатель женского рода в именах
- Суффикс принадлежности (1-е лицо)
- В корнях: *xha-* (разум), *sha-* (язык)

**Графическое строение (16×16):**
- Вертикальная черта с точкой в верхней части
- Минималистичный знак

**Связи:**
- → G-0005 (ī) — долгий вариант
- → L-000020 (xha-) — корень «разум»
- → L-000021 (sha-) — корень «язык»
