# G-0002 — Ā

**Транслитерация:** ā  
**Произношение:** /aː/  
**Тип:** Гласный (V), долгий  
**Слог:** aa

**Описание:** Долгий вариант G-0001. Маркирует длительность гласного, которая может менять значение слова.

**Употребление:**
- *khar-* (свет) vs *khār-* (ослепительный свет)
- В суффиксах множественного числа

**Графическое строение (16×16):**
- G-0001 + дополнительный горизонтальный элемент
- Удлиняющая черта снизу

**Связи:**
- ← G-0001 (a) — краткий вариант
