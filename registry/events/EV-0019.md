# EV-0019: Принятие Хартии Разумных Видов

**Тип:** реформа
**Дата:** +1810
**Эпоха:** Эпоха Интеграции
**Статус:** active
**Создан:** 2026-07-16

## Описание

Совет Единства и представители Neth'aari подписали Хартию Разумных Видов — первый межвидовой правовой документ в истории Xha'Ruun. Хартия гарантировала права обоих видов на самоопределение, территорию, культурную автономию и совместное использование ресурсов планеты. Документ был ратифицирован всеми городами-государствами.

## Причины

Рост числа контактов между кхарианцами и Neth'aari привёл к территориальным конфликтам в прибрежных зонах. Отсутствие правовой базы для межвидовых отношений создавало риск эскалации. Петиции Гильдии Ксенобиологов и общественное давление подтолкнули Совет к действию.

## Последствия

Формализация отношений между двумя цивилизациями. Создание Совместного Прибрежного Совета для разрешения споров. Открытие программ культурного обмена. Прецедент для будущих контактов с иными видами. Принятие Хартии считается одним из величайших дипломатических достижений Эпохи Интеграции.
