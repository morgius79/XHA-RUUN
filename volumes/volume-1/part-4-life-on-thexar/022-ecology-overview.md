# Глава 22: Экология Théxar — обзор биосферы

> *«Шесть ног бегут по равнине, четыре крыла рассекают небо, три глаза высматривают добычу. Théxar — это симфония жизни, где каждая партия написана для своего исполнителя.»*
> — Из «Энциклопедии живых форм», Институт Ксенобиологии Шан-Эн, +3 200 г. Э.Е.

---

## Введение

Биосфера Théxar — одна из богатейших в известной части галактики. Оценочно, на планете существует от 8 до 12 миллионов видов, из которых описано лишь ~2.8 миллиона (~28%). Для сравнения: на Земле описано ~1.8 млн видов из предполагаемых 8–10 млн.

Шесть конечностей — доминирующий план строения для крупных животных (аналог билатеральной симметрии на Земле). Однако эволюция Théxar породила и иные решения: четырёх- и восьмиконечные формы, радиально-симметричные организмы и псевдоподиальные гиганты.

Эта глава — обзор живого мира Théxar: трофическая структура, биогеографические царства, границы жизни и сравнение с Землёй.

### 22.1.3 Серный цикл

Уникальная особенность экосистем Théxar — **активный серный цикл**:

```
    СЕРНЫЙ ЦИКЛ THÉXAR

    ┌──────────────────────────────────────────────────┐
    │                                                  │
    │  ┌─────────┐    ┌──────────┐    ┌──────────┐   │
    │  │ H₂S     │───►│ Хемо-    │───►│ Органи-  │   │
    │  │ (атм.   │    │ синтез   │    │ ческие   │   │
    │  │ + вулк.)│    │ (S⁰ →    │    │ соеди-   │   │
    │  └────┬────┘    │ SO₄²⁻)   │    │ нения S  │   │
    │       │         └──────────┘    └────┬─────┘   │
    │       │          ┌──────────┐        │        │
    │       └─────────►│ Редуценты│◄───────┘        │
    │                  │ (H₂S ←)  │                │
    │                  └──────────┘                │
    └──────────────────────────────────────────────────┘
```

*Рис. 22.1. Серный цикл Théxar. H₂S — не токсин, а ключевой метаболический субстрат, замыкающий отдельный энергетический цикл.*

### 22.1.4 Основные биохимические циклы

| Цикл | Ключевые соединения | Отличие от Земли |
|------|--------------------|-----------------|
| Углеродный | CO₂, органические кислоты | +C₄-подобный фиксация |
| Азотный | NH₃, NO₃⁻, аминокислоты | **NH₃** как промежуточный |
| Серный | H₂S, S⁰, SO₄²⁻ | **Ключевая роль H₂S** |
| Фосфорный | PO₄³⁻, XTP, ДНК | P входит в P-нуклеотид |
| Кремниевый | SiO₂, Si-каналы | **Уникален** |
| Селеновый | Se, SeMet | Дополнительный антиоксидант |

## 22.2 Биогеографические царства

Théxar делится на **3 биогеографических царства** (соответствующих континентам), каждое с уникальным набором видов:

| Царство | Континент | Эндемизм | Ключевая особенность |
|---------|-----------|----------|---------------------|
| **Кхаракское** | Kharak | 45% | Наибольшее разнообразие, центр происхождения Xha'Ruun |
| **Рхувальское** | Rhuval | 38% | Адаптация к засушливому климату |
| **Вексарское** | Vexar | 52% | **Самый высокий эндемизм** (изоляция) |
| **Океаническое** | Оба океана | 35% | Морские виды, общие для всех континентов |

## 22.3 Границы жизни

### 22.3.1 Условия обитаемости

| Параметр | Минимум | Оптимум | Максимум |
|----------|---------|---------|----------|
| T | −12°C | +12…+38°C | +72°C |
| pH | 2.5 | 7.6–8.4 | 11.0 |
| Давление | 0.01 атм | 0.5–5 атм | 120 атм |
| Влажность | 10% | 50–95% | 100% |
| Радиация | 0 мЗв/год | до 100 мЗв/год | **~3 Зв/год** |

### 22.3.2 Экстремофилы

Как и на Земле, на Théxar существуют организмы, приспособленные к экстремальным условиям:

| Тип | Среда | Особенность |
|-----|-------|-------------|
| Термофилы | Гидротермальные источники (до +120°C) | Липиды с кремниевыми мостиками |
| Психофилы | Полярные регионы (−12…+2°C) | Аммиачный криопротектор |
| Ацидофилы | Вулканические озёра (pH 2.5–4.0) | Кремниевые каналы устойчивы |
| Радиофилы | Пустыни (высокий УФ) | Удвоенные системы репарации |
