# Приложение E: Глоссарий терминов Тома I

> Полный словарь терминов, используемых в Томе I: Вселенная. Формат: **термин** [язык] — определение. → см. соответствующую главу для контекста.

---

## A–Z (латиница)

**Khar'Vex** [Xha'Ruun] — звезда системы, оранжевый карлик K2V. «Подательница света». → см. [Гл.02]

**Kharak** [Xha'Ruun] — крупнейший континент Théxar (34.2 млн км²). «Земля света». → см. [Гл.03]

**Kharru** [Xha'Ruun] — одомашненное травоядное (аналог КРС). Шестиногое, класс Thermochromadae. → см. [Гл.05], [Гл.11]

**Rhuval** [Xha'Ruun] — второй по величине континент Théxar (21.7 млн км²). «Великая земля». → см. [Гл.03]

**Sha'Ruun** [Xha'Ruun] — язык Xha'Ruun. «Язык народа». → см. [Гл.06]

**Serrin** [Xha'Ruun] — крупнейший спутник Théxar (R=425 км). → см. [Гл.03]

**Thel** [Xha'Ruun] — малый спутник Théxar (R=178 км). → см. [Гл.03]

**Théxar** [Xha'Ruun] — планета, родина Xha'Ruun. «Земля-мать». → см. [Гл.03]

**Thalass** [Xha'Ruun] — крупнейший спутник Rhuun (R=2 840 км). Кандидат на колонизацию. → см. [Гл.02]

**Vexar** [Xha'Ruun] — третий континент Théxar (18.5 млн км²). «Творящая земля». → см. [Гл.03]

**Xha'Ruun** [Xha'Ruun] — разумный вид. «Мыслящий народ» (полная форма: Xha'Ruun-Vokh). → см. [Гл.05]

---

## А

**Аммиачная модификация** [биохим.] — присутствие ~3% NH₃ во внутриклеточной жидкости организмов Théxar. Криопротектор, модификатор растворимости. → см. [Гл.04], [Гл.18]

**Астеносфера** [геол.] — частично расплавленный слой верхней мантии Théxar (глубина 410–660 км). → см. [Гл.03]

---

## В

**Вексы** (vex-gran) [Xha'Ruun] — верхняя пара конечностей Xha'Ruun, эволюционно бывшие крыльями. → см. [Гл.05]

**Вексаний (Vx)** [хим.] — сверхтяжёлый элемент (Z=128, A=330). Плотность 24 г/см³. → см. [Гл.09]

---

## Г

**Гемоцианин** [биох.] — медьсодержащий пигмент крови Xha'Ruun, сине-зелёный цвет. → см. [Гл.05]

**Гидроксивалин (hVal)** [биох.] — уникальная аминокислота Théxar, обеспечивает термостабильность белков. → см. [Гл.04], [Гл.18]

**Гхарра** [геогр.] — Южный океан Théxar (87.5 млн км²). → см. [Гл.03]

---

## Д

**Дыхательная цепь Théxar** [биох.] — 5 комплексов (на 1 больше земной), эффективность на ~10% выше. → см. [Гл.04], [Гл.20]

---

## К

**Квартетный слух** [биол.] — способность определять направление звука с точностью ±0.5° благодаря 4 ушам. → см. [Гл.05]

**Кислородная катастрофа** [биохим.] — насыщение атмосферы O₂ фотосинтезом ~3.0 млрд л.н. Вымирание ~40% анаэробов. → см. [Гл.04], [Гл.21]

**Кремнеаспарагин (SiAsn)** [биох.] — уникальная аминокислота Théxar, содержит кремний. Служит для построения Si-каналов и структурных белков. → см. [Гл.04], [Гл.18]

**Кремнелипены** [биох.] — дополнительные пигменты растений Théxar, содержат кремний. Защитная функция. → см. [Гл.04], [Гл.20]

**Ксантинозин (X)** [биох.] — один из шести нуклеотидов ДНК Théxar. Пурин, образует тройную связь с пиримидином-N (P≡X). → см. [Гл.04], [Гл.19]

**Кхарний (Kh)** [хим.] — сверхтяжёлый элемент (Z=127, A=327). Ферромагнетик, сверхпроводник. Период полураспада: 12 400 лет. → см. [Гл.09]

---

## Н

**Нейтер (⚲)** [биол.] — третий пол Xha'Ruun. Стерильная особь (~65% популяции), рабочая функция. → см. [Гл.05]

**Нурий (Nur)** [хим.] — сверхтяжёлый элемент (Z=130, A=336). Катализатор. Период полураспада: 1 800 лет. → см. [Гл.09]

---

## П

**Пиримидин-N (P)** [биох.] — один из шести нуклеотидов ДНК Théxar. Уникальный пиримидин, образует тройную водородную связь с ксантинозином (X≡P). → см. [Гл.04], [Гл.19]

---

## С

**Селенометионин (SeMet)** [биох.] — уникальная аминокислота Théxar, содержит селен вместо серы. Антиоксидант, радиозащита. → см. [Гл.04], [Гл.18]

**Серный цикл** [экол.] — уникальный биогеохимический цикл Théxar, в котором H₂S замыкает отдельный энергетический поток. → см. [Гл.04], [Гл.11], [Гл.22]

---

## Т

**Тетрахроматия** [биол.] — цветовое зрение Xha'Ruun с 4 типами колбочек (RGB + УФ). → см. [Гл.05]

**Тироцианин (Tcn)** [биох.] — уникальная аминокислота Théxar. Ароматическая + CN-группа, обеспечивает электронный перенос. → см. [Гл.04], [Гл.18]

**Тринокулярное зрение** [биол.] — зрение Xha'Ruun с тремя глазами, высокоточная оценка глубины. → см. [Гл.05]

**Тхалас** [геогр.] — Северный океан Théxar (94.2 млн км²). → см. [Гл.03]

**Тхарий (Thr)** [хим.] — сверхтяжёлый элемент (Z=129). Тугоплавкий (T_пл = 4 200 K). → см. [Гл.09]

---

## Ф

**Фотосистема III** [биох.] — дополнительный фотосинтетический комплекс Théxar, поглощающий в УФ-A (320–380 нм). → см. [Гл.04], [Гл.20]

---

## Х

**Хиральность (D)** [биох.] — зеркальная по отношению к земной: D-аминокислоты, L-сахара. → см. [Гл.04], [Гл.18]

**Хромофилл** [биох.] — фотосинтетический пигмент Théxar (пик поглощения 570 нм). Обеспечивает сине-фиолетовый цвет листвы. → см. [Гл.04], [Гл.20]

---

## Ш

**Шестибуквенный код** [биох.] — генетический код Théxar из 6 нуклеотидов (G, C, A, T, P, X). 216 кодонов, 24 аминокислоты. → см. [Гл.04], [Гл.19]

---

## Э

**Электрорецепция** [биол.] — рудиментарная способность Xha'Ruun ощущать слабые электрические поля (0.1 мкВ/см). → см. [Гл.05]

**Эмоциональный резонанс** [псих.] — групповая синхронизация эмоций через феромонный канал. → см. [Гл.05]

---

## XTP

**XTP (ксантинозинтрифосфат)** [биох.] — основная энергетическая валюта клеток Théxar. Аналог земного АТФ, несёт ~15% больше энергии на молекулу. → см. [Гл.04], [Гл.20]
