Уровень 2 — Полуавтоматика

Если AI не уверен (confidence 60–90%), объявление уходит на ручную проверку.

Триггеры: жалобы пользователей, резкий рост бюджета (5×+ за день), пограничный контент, аккаунт без истории

Уровень 3 — Ручная модерация

Реальный сотрудник Meta смотрит креатив и ленд. Видит: объявление, посадочную страницу, историю аккаунта, жалобы.

2. Стратегия «Слоёный пирог» — 4 уровня защиты

🔹 Уровень 1 — Креатив
Нет стоп-слов, замаскированный визуал, эвфемизмы, правильный цветовой тон
🔹 Уровень 2 — Преленд
Невидим для модерации, показывает контент только реальным пользователям
🔹 Уровень 3 — Аккаунт + Прокси
Чистый фармленный акк, жирный прокси, антидетект-браузер, карта той же страны
🔹 Уровень 4 — Инфраструктура
BM с историей, несколько страниц, админы-резиденты, распределение рисков
Главный принцип: Если пробили уровень 1 — ловит уровень 2. Если пробили 2 — ловит 3. Задача — чтобы модератору было слишком сложно пробить все 4 уровня.

Техники маскировки креатива

  1. Размытие / шум — 2–5% шума на картинку, AI не распарсит, человек видит норм
  2. Реверсивная картинка — отзеркалить, сменить цвета, обрезать → не совпадает хеш с забаненными
  3. Текст поверх текста — пиши с опечатками или перекрывай элементами
  4. Видео вместо картинки — видео модерируется мягче, AI сложнее анализировать
  5. Смена формата каждые 2–3 дня — не дать AI накопить статистику

4. Уровень 2 — Защита преленда

Когда ты загружаешь объявление, бот Meta делает запрос на URL. Если видит казино-контент → бан.

Методы обхода

Geo-Targeting преленда (NGINX)

if ($http_user_agent ~* (facebook|meta)) {
    return 301 /clean.html;  # для бота — чистая страница
}
# для реального пользователя — преленд

IP-based блокировка

Заблокировать доступ для IP-диапазонов Meta. Преленд — только с "обычных" IP.

JS-редирект (проще, но менее надёжный)

if (!document.referrer.includes('facebook.com')) {
    window.location.href = '/clean.html';
}

Цепочка без прямого оффера

Meta → Преленд → Keitaro → Оффер. Meta видит только преленд, никогда — оффер.

Что должно быть на преленде для модератора

  • Нет стоп-слов (casino, bet, slots)
  • Нет упоминаний азартных игр
  • Нет кнопок "Играть", "Ставки"
  • Выглядит как обычный блог/статья
  • Есть политика конфиденциальности

Сигналы, ведущие к бану акка

  • Вход с необычного местоположения
  • Публичный IP / VPN, который уже банился
  • Быстрое добавление друзей
  • Резкая смена имени / фото
  • Несколько акков на одном IP

6. Уровень 4 — Инфраструктура

Business Manager

BM — это твой щит. Когда банят рекламный акк — BM должен остаться жив.

  • Не делай всё в одном BM — распределяй риски
  • На каждый BM: 2+ админа (резиденты страны)
  • BM с историей 3+ месяца живёт дольше
  • Подтверждённый бизнес (верификация) продлевает жизнь в разы
  • К BM привязана страница с историей и постами

Стратегии рекламных аккаунтов

Не создавай новую страницу — купи готовую с историей (1+ месяц, посты, подписчики).

7. Режимы запуска — от мягкого к жёсткому

🟢 Режим A — Безопасный (рекомендуется для старта)
  • Фармленный акк 60+ дней
  • Резидентный прокси
  • Антидетект-браузер
  • Преленд с гео-таргетингом
  • 1 акк → 1–2 кампании/день → $20–50/день
🟡 Режим B — Рабочий (баланс риска и профита)
  • Фармленный акк 30+ дней
  • Дата-центр IPv4 (чистый)
  • Антидетект-браузер
  • Преленд без жёстких стоп-слов
  • 1 акк → 3–5 кампаний → $50–100/день
🔴 Режим C — Агрессивный (высокий риск)
  • Свежие акки 7–14 дней
  • Дата-центр IPv4
  • Обычный браузер (чистый Chrome)
  • Преленд с JS-редиректом
  • $100–300/день — успеть отлиться до бана

Если аккаунт не вернули

  1. Сменить прокси (старый мог быть в чёрном списке)
  2. Сменить антидетект-профиль
  3. Купить новый аккаунт
  4. Проверить креатив и преленд на стоп-слова
  5. Запускаться через месяц, не сразу

9. Чек-лист перед запуском

Каждый запуск

  • Аккаунт фармленный (30+ дней)
  • Прокси жирный, страна = стране аккаунта
  • Антидетект настроен (fingerprint под страну)
  • Креатив проверен на стоп-слова
  • Текст объявления чистый
  • Преленд не показывается ботам Meta
  • Преленд не содержит стоп-слов
  • Ссылка идёт через Keitaro (не прямая)
  • Домен не засвечен в казино раньше
  • Карта = страна аккаунта
  • Бюджет на старте — минимальный (прогрев)

Еженедельно

  • Менять креативы (хотя бы раз в 3–4 дня)
  • Проверять живучесть прокси
  • Закупать новые аккаунты (запас)
  • Смотреть, какие креативы живут дольше

10. Сколько аккаунтов нужно

Исходные данные:
Средняя жизнь аккаунта: 14 дней
Бюджет на акк/день: $30
Время на фарминг (если сам): 30 дней
Расход: 1 акк живёт 14 дней → 2 акка в месяц на одну кампанию
Для стабильной работы: 3 акка в обороте (льёт, прогревается, запас)
Расход в месяц: 6–8 акков, $30–150 на акки
