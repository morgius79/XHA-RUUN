1. Структура рекламного кабинета Facebook

Иерархия

Личный профиль → Business Manager (BM) → Ad Account (РК) → Page (страница) → Pixel (пиксель)
СущностьРоль
Личный профильТвоё «лицо» в Facebook. К нему привязывается BM.
Business Manager (BM)Штаб-квартира. Внутри BM создаются рекламные кабинеты, страницы, пиксели.
Рекламный аккаунт (РК)То, где ты крутишь рекламу, видишь spend, статистику. Лимиты: от $50/день
Страница (Page)Объявление всегда идёт от имени страницы. Без страницы — нельзя запустить рекламу.
Пиксель (Pixel)JS-код на лендинге. Передаёт действия пользователя обратно в Facebook.

Business Manager

Ты обязательно работаешь через BM. Не через личный кабинет.

  • Создаёшь на business.facebook.com/overview
  • Внутри BM: 3-5 рекламных аккаунтов, 1-2 страницы, 1-2 пикселя
  • Лимиты нового BM: только 2 рекламных аккаунта, низкий spending limit
  • Лимиты повышаются временем и тратами

Рекламный аккаунт

  • Создаётся внутри BM, получает числовой ID
  • Spending limit: $50/день для нового. Повышается ступенями: $100 → $250 → $500+
  • Валюта: USD (при создании, потом не меняется)
  • Карты: лучше иметь 2-3 запасных

Страница

  • Для серой вертикали: страница должна выглядеть живой (аватар, 3-5 постов, описание)
  • Не используй одну страницу для всего — если её заблокируют, вся реклама упадёт

2. Прокси — подробно

Proxy — посредник между твоим компьютером и интернетом. Facebook видит IP прокси, а не твой реальный.

Правило: 1 аккаунт = 1 уникальный residential IP.

Типы прокси

ТипОткуда IPДля баинга?
ResidentialРеальные устройства людей по всему миру✅ Да
MobileС мобильных операторов✅ Да
DatacenterСервера в дата-центрах❌ Нет

Как купить (Proxy6)

  1. Регистрация: proxy6.net → Sign Up (email под баинг)
  2. Пополнение: USDT (TRC20), AdvCash, Capitalist — от $5-10
  3. Покупка: Аренда прокси → IPv4 → HTTP(S) → 1 шт → 1 месяц
  4. Формат: http://user:pass@ip:port
🧪 Проверка прокси: Создай профиль в антидетект-браузере → вставь прокси → открой 2ip.ru — должен показываться IP прокси, не твой. Проверь whoer.net на утечки (WebRTC, DNS).

Частые проблемы

СимптомРешение
Не подключаетсяПроверь формат, добавь http:// перед IP
МедленнаяЗапроси замену через поддержку Proxy6
Facebook видит твой IPWebRTC утекает — отключи в антидетекте

3. Keitaro — трекер

Зачем нужен

  • Считать клики + конверсии (Facebook не знает, кто зарегистрировался на казино)
  • Маскировать ссылку (Facebook видит твой-домен.com/click?id=...)
  • Оптимизировать: видеть какой креатив даёт профит

Как работает

Facebook объявление → ссылка на Keitaro → Keitaro записывает клик + страну + ОС + креатив → Редиректит на оффер казино → Казино конвертит → шлёт Postback в Keitaro → Keitaro показывает статистику

Установка

Вариант А — Облачная версия: Купить готовый хостинг Keitaro ($15-30/мес). Ничего не настраивать.

Вариант Б — Self-hosted (на своём VPS):

  1. Арендовать VPS (Hetzner CX22 — €7/мес)
  2. Установить Ubuntu 22.04
  3. Поставить панель (HestiaCP / CyberPanel)
  4. Направить домен (A-запись → IP сервера)
  5. Скачать Keitaro, закинуть в /var/www/html, запустить установщик

4. Facebook-аккаунты

Свой личный — не использовать

Если заблокируют — потеряешь всё: мессенджер, страницы, BM.

Вариант А — Купить фармленные (рекомендуется)

Фармленный аккаунт: создан 30+ дней назад, есть аватар, друзья (30+), посты, лайки, поведение живого человека.

  • Цена: $3-10 за штуку
  • В комплекте: Email + пароль, cookies, user-agent
  • Где искать: Telegram-каналы, чаты по арбитражу

Вариант Б — Фармить самому

  1. Создать аккаунт через антидетект + прокси
  2. Каждый день: лайкать, добавлять друзей, смотреть видео
  3. 2-4 недели — и аккаунт готов к использованию

Как заходить в купленный аккаунт (важно!)

  1. Создать новый профиль в антидетект-браузере
  2. Указать прокси той же страны, что и аккаунт
  3. Импортировать cookies (если продавец дал)
  4. Ввести логин/пароль
  5. Проверить whoer.net — не светится ли реальный IP
  6. Первые 1-2 дня: просто сидеть, лайкать, листать ленту

Сколько нужно для старта

Минимум 2-3 аккаунта: 1 основной + 1-2 запасных. Когда один забанят — быстро переключаешься.

5. Платёжные карты

Как работает оплата в Facebook

  1. Авторизация (pre-auth): при привязке снимает $1-2 и сразу возвращает
  2. Списание (billing): при достижении порога $25-50 или раз в 7 дней

Какие карты работают

Тип картыРаботает?
Твоя личная банковская карта❌ Чаще всего нет
Российские карты❌ Не работают
Виртуальные карты (Capitalist)⚠️ Бывает
Карты европейских/американских банков✅ Лучше всего

Рекомендация для старта

  1. Зарегистрироваться на Capitalist (capitalist.net)
  2. Пополнить через USDT
  3. Заказать виртуальную карту ($2-5 за выпуск)
  4. Привязать к рекламному кабинету Facebook
  5. Если не прошла — заказать другую (другой BIN)

BIN (первые 6 цифр карты): Facebook смотрит на него и определяет банк. Некоторые BIN-ы «в доверии», другие — нет.

Сколько карт: минимум 2-3. 1 основная + 1-2 запасных.

CRG

CRG (Capping Revenue Goal) — платят CPA не за 100% регистраций, а только если игрок сделал депозит. Пример: CPA $30, CRG 1:3 — платят за каждую 3-ю регистрацию с депозитом.

7. Связка Facebook → Keitaro → Оффер

Полная схема

Пользователь в Facebook → видит объявление → кликает ↓ [твой-домен.xyz/click?id=123&camp=test1] ↓ KEITARO: записывает клик, страну, ОС, креатив → редиректит ↓ [ссылка партнёрки / оффер казино] ↓ ЛЕНДИНГ КАЗИНО → регистрация / депозит ↓ ПАРТНЁРКА → Postback → KEITARO ↓ ТЫ: потратил $100 → 10 конверсий → заработал $300

Настройка оффера в Keitaro

  1. Offers → Add Offer: Name, Offer URL (ссылка партнёрки + {clickid})
  2. Action Type: 302 Redirect
  3. Parameters: Target URL, Country

Настройка потока (Stream)

  1. Streams → Add Stream: Name, Offer ID, Traffic Source, Countries
  2. Traffic Distribution: 100%

Настройка кампании в Keitaro

  1. Campaigns → Add Campaign: Name, Stream ID
  2. Unique Visits: 72 hours
  3. После сохранения получить Campaign URL — его вставляешь в Facebook

Postback (самое важное)

Postback — сигнал от партнёрки в Keitaro: «клик номер 123 дал конверсию». Без него Keitaro видит только клики, не зная кто конвертился.

  1. Keitaro → Offers → твой оффер → Postback URL
  2. Вставляешь URL в партнёрке (в настройках оффера)
  3. Партнёрка шлёт макросы: {clickid}, {payout}

Создание кампании в Facebook

  1. Ads Manager → Create → Campaign
  2. Цель: Traffic или Conversions
  3. Ad Set: Targeting по ГЕО, Budget $10-20/день
  4. Ad: В поле Website URL — ссылка из Keitaro, Display Link — безобидный домен

8.5 Структура креатива

Primary Text (основной текст):

❌ Плохо

«Получи 100% бонус на первый депозит в казино X! Самые популярные слоты!»

✅ Хорошо

«Недавно наткнулся на одну интересную платформу для аналитики. Решил протестировать — результаты удивили. Если интересно — пиши в личку, скину ссылку.»

Headline (заголовок, до 40 символов):

  • ❌ «Casino — лучшие слоты и бонусы»
  • ✅ «Неожиданный результат за 2 недели»
  • ✅ «Проверено на себе — делюсь опытом»

CTA (кнопка): Используй Learn More или Subscribe. Никогда Sign Up / Get Offer / Book Now.

Display Link: показывай безобидный домен (например plylg.xyz/trading-strategy), не plylg.xyz/casino.

Картинки: технические требования

  • Минимум 1080×1080 px (квадрат) или 1200×628 px
  • Текст на картинке — не больше 20% площади
  • Формат: JPG или PNG
  • Соотношения: 1:1 (квадрат) для старта, 4:5 (портрет) — выше CTR

Шрифты на картинке

  • Используй: Helvetica, Arial, Montserrat, Inter, SF Pro
  • Не используй: Comic Sans, Papyrus, рукописные
  • Размер: не меньше 24pt на исходнике 1080×1080
  • Цвет: белый с тенью или из палитры картинки

Цветовая палитра

ЦветАссоциацияПрименение
ЗелёныйДеньги, ростГрафики, цифры (+7%) — осторожно
СинийДоверие, надёжностьСамый безопасный
КрасныйОпасность, азартТолько акценты
Чёрный/БелыйПремиумФоны
Избегай комбинации красный + золотой + чёрный — это классический «казино-стиль», AI-модерация считывает.

8.6 Видео-креативы

Видео конвертит лучше картинок: CTR выше на 30-50%, CPC ниже на 20-40%.

Правило 3 секунд: первые 3 секунды решают, досмотрят ли видео.

  • Резкое движение, смена кадра, текст-интрига
  • Оптимальная длина: 15-30 секунд
  • Без звука (большинство смотрит без звука) — обязательны субтитры
  • Формат: квадрат 1:1 или портрет 4:5

Варианты видео для гемблинга:

ТипОписаниеСложность
Слайд-шоу3-5 картинок, текстовая озвучка🟢 Просто (CapCut)
Экранная записьScreen record приложения/сайта🟡 Средне
AI-видеоСгенерировано Runway/Kling🟡 Средне

Субтитры (обязательно):

  • Белый текст с чёрной обводкой
  • 2-3 слова на кадр
  • Шрифт: без засечек (Helvetica, Arial)

Слайд-шоу за 5 минут

  2. Добавить переходы (Slide/Fade, 0.3-0.5 сек)

12. Диагностика: креатив не взлетел

Пошаговый разбор

Шаг 0: Собери данные. Для значимого результата нужно: 500+ показов (CTR), 50+ кликов (CPC), 100+ кликов (CR).

Шаг 1 — CTR < 0.5%:

  • Скучная картинка → смени визуал
  • Текст нечитаем на картинке → увеличь шрифт
  • Непонятно о чём объявление → упрости месседж
  • Слишком рекламно → пиши как человек
  • Frequency > 3 → смени креатив/аудиторию

Шаг 2 — CPC выше среднего:

  • Высокая конкуренция → сузь таргетинг
  • Relevance Score < 6 → смени креатив
  • Дорогой placement → отключи Audience Network

Шаг 3 — Клики есть, конверсий 0:

  • Сломана цепочка Keitaro → проверь редирект
  • Обманутые ожидания → согласуй креатив с лендингом
  • Лендинг не грузится на мобильных → проверь
  • Лендинг не на том языке → проверь ГЕО
  • Партнёрка не шлёт постбэк → проверь настройки

Диагностическая таблица

СимптомПроблемаЧто менять
Много показов, мало кликов (CTR <0.5%)Креатив не цепляетКартинку + текст
Клики есть, конверсий 0ЦепочкаKeitaro / лендинг / оффер
Клики дорогие (CPC выше рынка)Таргетинг / RelevanceАудиторию или креатив

Жёсткие критерии отключения

  1. Потрачено $50 — 0 конверсий → отключай
  2. CTR < 0.3% при 2000+ показов → отключай
  3. CPA > payout × 3 → отключай
  4. Frequency > 5 при 0 конверсиях → отключай
  5. Креатив отклонён модерацией → не перезапускай

Если всё отключил — что дальше?

  1. Смени угол подачи (было «инвестиции» → стало «тест/квиз»)
  2. Смени визуал (было лицо → стало графика/аналитика)
  3. Смени текстовую маскировку
  4. Смени аккаунт (если старый получил страйки)
  5. Если и это не работает — смени ГЕО или оффер

13.1 Что такое n8n и установка

n8n — open-source платформа оркестрации workflow. Визуальный редактор, где соединяешь ноды. Self-hosted — все данные на твоём сервере.

Почему n8n, а не Make.com/Zapier:

  • Self-hosted — данные не утекают
  • Бесплатный (нет лимита на задачи)
  • Любые HTTP-запросы
  • Работает с файлами (картинки, видео)
  • Можно писать JavaScript внутри нод

from flask import Flask, request, send_file
import subprocess, os, json, requests, io, time

app = Flask(__name__)

@app.route('/merge', methods=['POST'])
def merge_videos():
    data = request.json
    video_urls = data.get('video_urls', [])
    output_name = data.get('output_name', 'merged.mp4')

    with open('/tmp/urls.txt', 'w') as f:
        for url in video_urls:
            f.write(f"file '{url}'\n")

    cmd = ['ffmpeg', '-y', '-f', 'concat', '-safe', '0', '-i', '/tmp/urls.txt',
           '-c:v', 'libx264', '-preset', 'fast', '-crf', '23', f'/tmp/{output_name}']
    subprocess.run(cmd, check=True, capture_output=True)
    return send_file(f'/tmp/{output_name}', mimetype='video/mp4')

    output = f"/tmp/text_{int(time.time())}.mp4"

    cmd = ['ffmpeg', '-y', '-i', video_url,
           '-vf', f"drawtext=text='{text}':fontcolor=white:fontsize=36:x=(w-text_w)/2:y=h-100:shadowcolor=black:shadowx=2:shadowy=2",
           '-c:v', 'libx264', '-preset', 'fast', '-crf', '23', output]
    subprocess.run(cmd, check=True, capture_output=True)
    return send_file(output, mimetype='video/mp4')

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5001)

16. Claude-стратег с аналитикой

Архитектура

Facebook Marketing API → Python fb_stats.py → n8n (контекст) → Claude API (анализ + решение) → Фабрика генерирует креативы по новой стратегии → Через 24ч — новый цикл

Python-скрипт выгрузки статистики (fb_stats.py)

#!/usr/bin/env python3
import requests, json, os
from datetime import datetime, timedelta

ACCESS_TOKEN = os.getenv('FB_ACCESS_TOKEN')
AD_ACCOUNT_ID = os.getenv('FB_AD_ACCOUNT_ID')
OUTPUT_FILE = '/assets/analytics/fb_stats.json'

yesterday = (datetime.now() - timedelta(days=1)).strftime('%Y-%m-%d')
today = datetime.now().strftime('%Y-%m-%d')

url = f'https://graph.facebook.com/v18.0/act_{AD_ACCOUNT_ID}/insights'
params = {
    'fields': 'campaign_name,ad_name,impressions,clicks,spend,ctr,cpc,actions',
    'level': 'ad',
    'time_range': json.dumps({'since': yesterday, 'until': today}),
    'access_token': ACCESS_TOKEN,
}
data = requests.get(url, params=params).json()

campaigns = []
for ad in data.get('data', []):
    conversions = sum(int(a.get('value', 0)) for a in ad.get('actions', []) if a.get('action_type') == 'registration')
    campaigns.append({
        'ad_name': ad.get('ad_name', ''),
        'impressions': int(ad.get('impressions', 0)),
        'clicks': int(ad.get('clicks', 0)),
        'spend': float(ad.get('spend', 0)),
        'ctr': float(ad.get('ctr', 0)),
        'cpc': float(ad.get('cpc', 0)),
        'conversions': conversions,
        'cpa': round(float(ad.get('spend', 0)) / conversions, 2) if conversions > 0 else None
    })

with open(OUTPUT_FILE, 'w') as f:
    json.dump(campaigns, f, indent=2)
