# Xha'Ruun 🜁 GENESIS PACK

**Версия:** v0.3.0 — Genesis  
**Статус:** Наполнение завершено — сборка тома  
**Дата:** 2026-07-17

---

**Xha'Ruun** — научно-фантастическая вселенная, цивилизация, язык и многотомная энциклопедия. Проект ведётся как **software-репозиторий**: с семантическим версионированием, инструментами сборки, тестами канона и CI.

## Что внутри

```
GENESIS_PACK/
├── 00_START_HERE/     ← Вы здесь
├── 01_WORLD/          # Вселенная, космология, планета
├── 02_LANGUAGE/       # Язык Xha'Ruun
├── 03_WRITING/        # Письменность и графемы
├── 04_HISTORY/        # Хронология и история
├── 05_REGISTRY/       # Реестры (графемы, лексемы, правила…)
├── 06_TOOLS/          # Инструменты и SDK
├── 07_ART/            # Визуальный канон и иллюстрации
├── 08_EXPORT/         # Сборка и форматы экспорта
└── 09_APPENDIX/       # Приложения
```

## Быстрый старт

```bash
# Сборка полной энциклопедии (Markdown)
npm run build:md

# Сборка PDF
npm run build:pdf:sha

# Валидация ссылок канона
npm run validate

# Генерация шрифта Sharuu
npm run build:font
```

## Принципы

1. **Внутренняя согласованность** — каждый факт не противоречит ни одному другому.
2. **Неизменяемость канона** — аксиомы меняются только через повышение версии.
3. **Единый источник истины** — канон хранится в этом репозитории.
4. **Тестируемость** — каждый факт может быть проверен инструментами validate.
