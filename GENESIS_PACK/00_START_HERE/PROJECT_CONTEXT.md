## 1. Философия проекта

### 1.1. Что такое Xha'Ruun

**Xha'Ruun** (кс'а-РУУН, 𐤑𐰀𐰺𐰆𐰣) — это:

1. **Научно-фантастическая вселенная** — проработанный мир с собственной физикой, астрономией, биологией и историей.
2. **Цивилизация** — пост-сингулярное общество трёх разумных видов на планете Théxar.
3. **Язык** — полноценный конланг с фонетикой, грамматикой, письменностью и лексиконом.
4. **Многотомная энциклопедия** — 10 томов, ~400 глав, охватывающих все аспекты мира.
5. **Software-проект** — репозиторий с версионированием, CI, тестами, инструментами сборки и SDK.

### 1.2. Ключевые принципы

1. **Внутренняя согласованность** — каждый факт не противоречит ни одному другому факту канона. Противоречие — баг, который исправляется.
2. **Неизменяемость канона** — аксиомы меняются только через повышение версии. История изменений обязательна.
3. **Единый источник истины** — канон хранится в этом репозитории. Внешние копии — справочные, не авторитетные.
4. **Тестируемость** — каждый факт может быть проверен инструментами validate.
5. **Итеративность** — проект растёт как software: версия v0.1 → v0.2 → ... → v10.0. Каждая версия — работающий артефакт.

### 1.3. Почему software-подход

Традиционные вселенные (вроде Dune, Star Wars, Trek) создаются как литературные Projects — с неизбежными противоречиями между книгами, авторами и эпохами. Xha'Ruun с самого начала строится как **инженерная конструкция**:

- Каждый факт — это строка в базе данных канона
- Каждая глава — это модуль с чётким интерфейсом (ссылки на другие главы)
- Каждый реестр — это таблица с валидацией
- Каждый артефакт (PDF, шрифт, карта) — это результат сборки
- Противоречия — это баги, которые обнаруживаются тестами

### 1.4. Отличие от традиционных worldbuilding-проектов

| Аспект | Традиционный подход | Xha'Ruun подход |
|--------|--------------------|-----------------|
| Версионирование | Отсутствует | SemVer (v0.1.0 → v10.0.0) |
| Согласованность | «Авторский произвол» | Автоматическая проверка |
| Контент | Линейные книги | Модульные главы с перекрёстными ссылками |
| Инструменты | Word / Scrivener | Git, CLI, валидаторы, сборщики |
| Коллаборация | Один автор | Репозиторий + CI + агенты |
| Артефакты | Только книга | PDF, шрифт, веб, SDK |

Каркас проекта. Содержит:

## 3. Вселенная Xha'Ruun

### 3.1. Место в реальности

Вселенная Xha'Ruun (самоназвание: **Vokh-Rass** — «Великая Гармония») — это пространство, законы физики которого на 97% тождественны законам нашей вселенной. Оставшиеся 3% — ключевые отличия, которые сформировали уникальный путь развития жизни и цивилизации.

### 3.2. Фундаментальные константы

| Константа | Значение | Отличие от Земли |
|-----------|----------|-----------------|
| Скорость света (c) | 2.998 × 10⁸ м/с | = |
| Гравитационная постоянная (G) | 6.674 × 10⁻¹¹ | = |
| Постоянная тонкой структуры (α) | 1/137.036 | = |
| Сильное взаимодействие (αₛ) | 0.1197 | **+0.0005** |
| Масса эл. нейтрино m(νₑ) | 0.12 эВ/с² | **×2** |
| Плотность тёмной материи (Ωᵈᵐ) | 0.262 | **+1.5%** |

**Ключевое отличие:** повышенная константа сильного взаимодействия привела к большему содержанию дейтерия в первичном нуклеосинтезе (в 3.2× больше, чем в нашей вселенной). Это имело каскад последствий для звёздной эволюции, планетарной геологии и — в конечном счёте — для технологического развития цивилизации.

### 3.3. Космология

| Параметр | Значение |
|----------|----------|
| Возраст вселенной | 13.82 млрд лет |
| Глобальная геометрия | Плоская (Ω ≈ 1.00) |
| Галактика | Спиральная с перемычкой (SBbc) |
| Диаметр галактики | ~105 000 св. лет |
| Положение системы | Спиральный рукав, ~26 000 св. лет от центра |
| Сверхскопление | Ланиакея |

Вселенная пространственно-временных измерений 3+1. Сверхсветовая передача информации невозможна — принцип причинности абсолютен.

### 3.4. Звёздная система Khar'Vex

**Звезда Khar'Vex** («Подательница света») — оранжевый карлик спектрального класса K2V.

| Параметр | Значение |
|----------|----------|
| Масса | 0.82 M☉ |
| Радиус | 0.78 R☉ |
| Светимость | 0.41 L☉ |
| Температура | 5 050 K |
| Возраст | 6.5 млрд лет |
| Металличность | [Fe/H] = +0.12 |
| Ожидаемая продолжительность | ~22 млрд лет |

Почему K2V, а не G2V (как Солнце):
- В 2.5× дольше жизни на главной последовательности
- Стабильнее светимость, меньше вспышек
- Пик излучения ~570 нм (оранжевый) — оптимален для фотосинтеза
- Обитаемая зона ближе к звезде (0.5–0.8 а.е.)

## 4. Планета Théxar

### 4.1. Основные параметры

Théxar (произн. /ˈθɛk.sɑr/) — дом цивилизации Xha'Ruun.

| Параметр | Значение | Отн. Земли |
|----------|----------|------------|
| Масса | 5.49 × 10²⁴ кг | 0.92× |
| Радиус | 6 112 км | 0.96× |
| Гравитация | 9.52 м/с² | 0.97× |
| Атмосфера | N₂ 71.8%, O₂ 22.6%, Ar 2.9%, CO₂ 0.8% | ≈ |
| Давление | 1.03 атм | = |
| Средняя T | +12°C | +2°C |
| Вода | 67% поверхности | 71% |
| Спутники | 2 (Serrin, Thel) | 1 |
| Сутки | 27.3 часа | 24 |
| Год | 412.7 местных суток | 365 |

### 4.2. Климат и география

Золотисто-бронзовое небо Théxar — результат спектра Khar'Vex и состава атмосферы. Планета получает на 42% меньше освещённости, чем Земля, что компенсируется повышенным CO₂ (+0.8%) и водяным паром — парниковый эффект +31°C.

**Континенты:**
- **Kharak** (34.2 млн км²) — северный, крупнейший
- **Rhuval** (21.7 млн км²) — южный
- **Vexar** (18.5 млн км²) — восточный
- Крупные острова (4.8 млн км²)

**Океаны:** Thalas (Северный, 94.2 млн км²) и Gharra (Южный, 87.5 млн км²).

**Высшая точка:** гора Verksan (7 820 м) на хребте Khar-Tor.

### 4.3. Биохимия

Жизнь Théxar построена на уникальной молекулярной основе:

- **6-буквенный генетический код** — G, C, A, T, P (пиримидин-N), X (ксантинозин) вместо земных 4
- **24 протеиногенные аминокислоты** вместо 20
- Противоположная хиральность (D-аминокислоты, L-сахара)
- Кремний-углеродные каналы в клеточных мембранах
- H₂S играет роль в энергетическом обмене (хемоавтотрофия)

Следствие: белковая сложность на 30–40% выше земной, метаболическая гибкость выше, радиационная стойкость в 5× выше.

## 5. Цивилизация

### 5.1. Разумные виды

На Théxar эволюционировали три разумных вида, объединённых в Союз Xha'Ruun (Xha'Ruun Covenant):

#### 5.1.1. Xha'ri (основной вид)
| Параметр | Значение |
|----------|----------|
| Рост | 2.2–2.4 м |
| Масса | 95–130 кг |
| Конечности | 6 (2 нижних, 4 верхних) |
| Глаза | 3 (тринокулярное зрение, 320–780 нм) |
| Уши | 4 (8 Гц – 42 кГц) |
| Сердец | 2 (грудное + брюшное) |
| Лёгких | 4 |
| Продолжительность жизни | 150–200 местных лет |
| Размножение | Яйцекладущие, 3 пола (♂ ♀ ⚲) |
| Цвет кожи | Золотисто-бронзовый до медно-красного |

Xha'ri — теплокровные гексаподы с тетрахроматическим зрением и электрорецепцией.

#### 5.1.2. Kel'vash
- Кремний-органическая биохимия
- Среда: высокогорья и пустыни
- Роль: инженеры, строители, архитекторы
- Численность: ≈ 1.2 млрд

#### 5.1.3. Moryn
- Водные разумные (океаны Théxar)
- Среда: глубокие океанические впадины
- Роль: биологи, океанологи, историки
- Численность: ≈ 0.8 млрд

### 5.2. Общество

**Форма правления:** Совет Единства — представительный орган из 49 членов, избираемых гильдиями и регионами.

**Административное деление:** 8 Доменов → Провинции → Округа.

**Гильдии (12):** Учёных, Инженеров, Медиков, Аграриев, Строителей, Торговцев, Хранителей, Воинов, Художников, Шахтёров, Навигаторов, Писцов.

**Социальная мобильность:** высокая, меритократия. Формально каст нет, де-факто стратификация по гильдиям.

**Население:** ≈ 2.8 млрд (все виды), урбанизация ~62%.

**Столица:** Khar-Vexar (18.6 млн) — крупнейший город, местоположение Совета Единства.

### 5.3. Законы (Аксиомы общества)

1. **Закон Единой Письменности** — все языки Союза записываются системой письма Xha'Ruun.
2. **Закон Согласия** — ни одно решение не принимается без консультации всех трёх видов.
3. **Закон Тишины** — зоны планеты объявлены заповедниками без технологий.
4. **Закон Знания** — каждый гражданин имеет доступ к любому факту канона.
5. **Закон Неприкосновенности Истории** — запрещено уничтожать или переписывать записи.
6. **Заповедь Роста** — численность регулируется сознательно (не более 0.3% в год).

### 5.4. Технологический уровень

**Тип:** ~0.78 по шкале Кардашёва (переход к Типу I).

| Технология | Статус |
|------------|--------|
| Управляемый термояд | Промышленный (тория + He-3) |
| Орбитальный лифт | 1 построен, 2 в строительстве |
| Маглев-транспорт | Повсеместно |
| Квантовая связь | Шифрованная, коммерческая |
| Био-оптические вычисления | Гибрид (электроника + био) |
| Межзвёздные перелёты | Не освоены (нет FTL) |
| Искусственный интеллект | Ограниченный (без самосознания — философский запрет) |
| Противогравитация | Считается невозможной |

**Фундаментальные ограничения:** скорость света абсолютна, FTL невозможен, путешествия во времени запрещены принципом причинности.

### 5.5. Энергия и экономика

- **Основной источник:** термоядерный синтез (торий + гелий-3)
- **He-3 добыча:** с поверхности спутника Thel и синтез
- **Дополнительно:** солнечные орбитальные станции, геотермальная, гидроэнергия
- **Экономика:** пост-дефицитная по базовым ресурсам; гильдийная организация труда

## 6. Язык и письменность

### 6.1. Язык Xha'Ruun

**Тип:** агглютинативный, SOV (субъект-объект-глагол).  
**Падежи:** 14.  
**Числа:** единственное, двойственное, множественное.  
**Времена:** прошлое, настоящее, будущее + вневременное.  

**Фонетика:** 24 согласных, 8 гласных. Ударение на первый слог корня. Фонотактика: (C)(C)V(C)(C).

**Эвиденциальность** — обязательная грамматическая категория. Каждое высказывание маркирует источник информации: прямое свидетельство, косвенное, инферентив (вывод), репортатив (пересказ).

### 6.2. Письменность (Sharuun)

**Тип:** смешанная слоговая (силлабарий + логограммы).  
**Направление:** справа налево, сверху вниз.  
**Базовых знаков:** 124 слоговых + 48 логограмм.  
**Графемы:** вписываются в модульную сетку 16×16.

Шрифт Sharuu генерируется через `npm run build:font` — SVG-шрифт на основе канонического набора графем.

### 6.3. Примеры корневых слов

| Корень | Значение |
|--------|----------|
| xa- | жизнь |
| xha- | разум, мысль |
| ruun- | народ, сообщество |
| khar- | свет, звезда |
| the- | земля, планета |
| sha- | язык, слово |
| ven- | истина, знание |

**Правила образования имён:** двусоставные, [основа-1]-[основа-2]. Ударение на второй части. Женские: -a, -ae. Мужские: -en, -un. Нейтеры: -in, -el.

## 9. Реестры и ID

### 9.1. Типы объектов

| Тип | Префикс | Формат ID | Диапазон |
|-----|---------|-----------|----------|
| Графема | G | G-0001 — G-0064 | 64 базовых знака |
| Лексема | L | L-000001 — L-999999 | Слова и морфемы |
| Правило | RULE | RULE-0001 — RULE-9999 | Грамматические и канонические |
| Искусство | ART | ART-000001 — ART-999999 | Иллюстрации, карты, арты |
| Локация | LOC | LOC-0001 — LOC-9999 | Географические точки |
| Персонаж | CH | CH-0001 — CH-9999 | Личности |
| Событие | EV | EV-0001 — EV-9999 | Исторические события |

### 9.2. Принципы регистрации

1. Уникальный ID присваивается один раз и не меняется.
2. Удалённые объекты — `retired`. ID не переиспользуется.
3. Ссылки на объекты всегда через ID, а не через названия.
4. Каждый ID имеет карточку: описание, статус, версия, ссылки.

### 9.3. Структура реестра

```
registry/
├── glyphs/
│   ├── G-0001.md
│   ├── G-0002.md
│   └── ...
├── lexemes/
│   ├── L-000001.md
│   └── ...
├── rules/
├── locations/
├── characters/
├── events/
└── art/
```

## 11. Правила разработки

### 11.1. Процесс

1. **Изучить канон** — перед добавлением любого факта проверить CANON.md.
2. **Зарегистрировать** — новый объект получает ID в соответствующем реестре.
3. **Написать** — контент в markdown с перекрёстными ссылками.
4. **Проверить** — `npm run validate` на согласованность.
5. **Закоммитить** — один коммит = одна логическая единица.

### 11.2. Формат коммитов

```
[том/реестр/инструмент] Краткое описание
```

Примеры:
```
[registry] Added G-0032 to G-0048
[volume-1] Chapter 007 – Atmospheric composition
[canon] Added Law of Silence clarification
[tooling] PDF builder: fixed page-break issue
```

### 11.3. Версионирование

**SemVer** (семантическое версионирование):
- **Мажорная** — нарушение обратной совместимости (изменение аксиомы)
- **Минорная** — добавление фактов, глав, инструментов
- **Патч** — исправление ошибок, опечаток, неточностей

### 11.4. Запрещено

1. Изменять CANON.md без обоснования и повышения версии.
2. Создавать факты без регистрации в реестре.
3. Переиспользовать ID удалённых объектов.
4. Вносить изменения, нарушающие согласованность с каноном.
5. Добавлять контент «на вырост» — только то, что нужно сейчас.

### 11.5. Проверка согласованности

Каждый новый факт проверяется на:
- Не противоречит ли прямой аксиоме канона
- Не противоречит ли косвенно (через цепочку логических выводов)
- Ссылается ли на существующие объекты по ID
- Корректны ли перекрёстные ссылки

## 12. Визуальный канон

### 12.1. Цветовая палитра

| Цвет | HEX | Применение |
|------|-----|------------|
| Золото | `#c9a84c` | Акцент, заголовки |
| Лазурь | `#5b7db5` | Ссылки, ID объектов |
| Бронза | `#8b6f4e` | Второстепенные элементы |
| Фон (тёмный) | `#0a0e17` | Экранная версия |
| Фон (светлый) | `#f5f0eb` | Печатная версия |

### 12.2. Шрифты

- **Основной:** Inter / IBM Plex Serif
- **Моноширинный:** JetBrains Mono / Fira Code
- **Sharuu:** Собственный SVG-шрифт (генерируется)

### 12.3. Графемы

Каждая графема вписывается в квадрат 16×16 единиц. Базовая линия: 12 единиц от верхнего края. Толщина штриха: 1 ед. (основной), 0.5 ед. (дополнительный). Формат: оптимизированный SVG.

### 12.4. Формат книг

Печатная версия: A5 (148×210 мм), поля 15 мм, колонтитулы с названием тома и номером главы. Оглавление — автоматическое, двухуровневое.
