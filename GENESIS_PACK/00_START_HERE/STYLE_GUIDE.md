# STYLE_GUIDE.md — Визуальный стиль и оформление Xha'Ruun

> Версия: 0.1.1  
> Подчиняется канону (CANON.md).

---

## 1. Шрифты

### 1.1. Основной шрифт (латиница/кириллица)
- **Заголовки:** Inter, Montserrat, или System UI sans-serif
- **Текст:** Inter, IBM Plex Serif, или System UI
- **Моноширинный:** JetBrains Mono, Fira Code

### 1.2. Шрифт Sharuu (письменность Xha'Ruun)
- Формат: SVG-шрифт (генерируется через `npm run build:font`)
- Название: **Sharuu Display** / **Sharuu Text**
- Все графемы вписываются в модульную сетку (см. раздел 3)
- Кегль: 12pt для текста, 18pt+ для заголовков

## 2. Цветовая схема

### 2.1. Основная палитра

| Название | HEX | Применение |
|----------|-----|------------|
| Фон (тёмный) | `#0a0e17` | Основной фон документации |
| Фон (светлый) | `#f5f0eb` | Печатная версия |
| Текст | `#e8e4d8` | Основной текст (тёмная тема) |
| Текст (печать) | `#1a1a1a` | Основной текст (светлая тема) |
| Акцент | `#c9a84c` | Золото — заголовки, ключевые термины |
| Акцент 2 | `#5b7db5` | Лазурь — ссылки, ID |
| Акцент 3 | `#8b6f4e` | Бронза — второстепенные элементы |
| Предупреждение | `#c94c4c` | Красный — нарушения канона |
| Успех | `#4caf50` | Зелёный — валидация пройдена |

### 2.2. Использование цветов в документации

```markdown
<!-- Заголовки первого уровня — с золотым акцентом -->
# Заголовок — используем акцентный цвет

<!-- Ссылки на ID — лазурные -->
см. [G-0042](05_REGISTRY/glyphs/g-0042.md)

<!-- Предупреждения о неканоничности — красный блок -->
> ⚠ **Неканонично:** описание противоречит CANON.md §3.2
```

### 2.3. Графемы и модульная сетка

- Каждая графема вписывается в квадрат 16×16 единиц
- Базовая линия: 12 единиц от верхнего края
- Толщина штриха: 1 единица (основной), 0.5 (дополнительный)
- Скругление: 0.25 единицы
- Формат экспорта: SVG, оптимизированный

## 3. Оформление документов

### 3.1. Заголовки

```markdown
# Том N: Название (H1 — только для томов и крупных разделов)

## Глава N: Название (H2 — для глав)

### Раздел (H3 — для подразделов)

#### Подраздел (H4 — для деталей)
```

### 3.2. ID объектов

Всегда в формате:

```
**G-0042** — графема «Xha» (свет/знание)
**L-000127** — лексема «xha'ruun» (цивилизация)
**RULE-0007** — Закон Единой Письменности
```

### 3.3. Формат карточек реестра

Все карточки реестра следуют единому шаблону:

```markdown
# ID: Название

**Тип:** тип_объекта
**Статус:** active | retired | draft
**Создан:** YYYY-MM-DD

## Описание

...

## Связанные объекты

- ID
```

Поля обязательны: ID, Тип, Статус, Создан, Описание.

### 3.4. Каноничные блоки

```markdown
> **Канон §3.2:** Планета Théxar — землеподобный мир массой 0.92 M⊕.
```

### 3.5. Предупреждения

```markdown
> ⚠ **Черновик:** Этот раздел не прошёл валидацию канона.

> 📝 **Примечание:** Этот факт требует согласования с Томом IV.

> ✅ **Проверено:** Соответствует CANON.md v0.2.0.
```

## 4. Иллюстрации

### 4.1. Форматы
- **Вектор:** SVG (предпочтительно)
- **Растр:** PNG (только для сложных текстур)
- **3D:** Не используется в Genesis Pack

### 4.2. Карты
- Масштабная сетка с указанием единиц
- Подписи на двух языках (русский + Xha'Ruun транслитерация)
- Условные обозначения внизу карты

### 4.3. Схемы графем
- Модульная сетка 16×16
- Подпись: ID графемы, транслитерация, значение

---

## 5. Вёрстка (PDF)

### 5.1. Параметры страницы
- Формат: A5 (148×210 мм) — для печатной версии
- Поля: 15 мм со всех сторон
- Колонтитулы: название тома слева, номер главы справа

### 5.2. Оглавление
- Автоматически генерируемое
- С номерами страниц
- Двухуровневое (том → глава)

### 5.3. Колонтитулы
```
[Том I: Вселенная] ──── 47 ──── [Глава 3: Планета]
```

---

## 6. Правила для рисунков

1. Каждый рисунок имеет ID (ART-XXXXXX).
2. Рисунок сопровождается подписью на русском.
3. Если применимо — подпись на языке Xha'Ruun.
4. Все рисунки регистрируются в реестре ART.
5. Векторные иллюстрации в едином стиле: минимализм, геометрия, золото-лазурная палитра.
