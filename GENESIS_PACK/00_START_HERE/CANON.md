## 2. Определение проекта

Xha'Ruun — научно-фантастическая вселенная, цивилизация, язык и многотомная энциклопедия.  
Проект ведётся как **software-репозиторий**: с версионированием, CI, тестами и инструментами сборки.

### 2.1. Принципы

1. **Внутренняя согласованность** — каждый факт не противоречит ни одному другому факту канона.
2. **Неизменяемость канона** — аксиомы меняются только через повышение версии.
3. **Единый источник истины** — канон хранится в этом репозитории. Внешние копии — справочные.
4. **Тестируемость** — каждый факт может быть проверен инструментами validate.

## 3. Вселенная

### 3.1. Название

- Полное: **Xha'Ruun** (кс'а-РУУН) / **Vokh-Rass** («Великая Гармония» — самоназвание вселенной)
- Символ: 𐤑𐰀𐰺𐰆𐰣
- Этимология: Древний корень *xha-* («свет/знание») + *ruun* («народ/сообщество»)

### 3.2. Фундаментальные константы

| Константа | Значение | Отличие от Земли |
|-----------|----------|-----------------|
| Скорость света (c) | 2.998 × 10⁸ м/с | = |
| Гравитационная постоянная (G) | 6.674 × 10⁻¹¹ | = |
| Постоянная Планка (h) | 6.626 × 10⁻³⁴ Дж·с | = |
| Постоянная тонкой структуры (α) | 1/137.036 | = |

| Плотность тёмной материи (Ωᵈᵐ) | 0.262 | **+1.5%** |

**Ключевое следствие:** избыток дейтерия (×3.2) в первичном нуклеосинтезе; галактики формировались на ~50 млн лет раньше.

### 3.3. Космология

- **Возраст вселенной:** 13.82 млрд лет
- **Размерность:** 3+1 (пространство + время)
- **Глобальная геометрия:** Плоская (Ω ≈ 1.00)
- **Принцип причинности:** Абсолютен — сверхсветовая передача информации невозможна
- **Галактика:** Спиральная с перемычкой (SBbc), ~105 000 св. лет
- **Система:** В спиральном рукаве, ~26 000 св. лет от центра
- **Сверхскопление:** Ланиакея

Жизнь Théxar базируется на уникальной молекулярной основе:

- **6-буквенный генетический код:** G, C, A, T, P (пиримидин-N), X (ксантинозин)
- **24 протеиногенные аминокислоты** (вместо 20)
- **Противоположная хиральность:** D-аминокислоты, L-сахара
- **Кремний-углеродные каналы** в клеточных мембранах
- **H₂S-зависимый энергообмен** (хемоавтотрофия широко распространена)
- Радиационная стойкость в **5× выше** земной

### 4.3. Законы (Аксиомы общества)

1. **Закон Единой Письменности** — все языки Союза записываются системой письма Xha'Ruun.
2. **Закон Согласия** — ни одно решение не принимается без консультации всех трёх разумных видов.
3. **Закон Тишины** — определённые зоны планеты объявлены заповедниками без технологий.
4. **Закон Знания** — каждый гражданин имеет доступ к любому факту канона.
5. **Закон Неприкосновенности Истории** — запрещено уничтожать или переписывать исторические записи.
6. **Заповедь Роста** — численность популяции регулируется сознательно, не более 0.3% прироста в год.

## 7. Реестры

### 7.1. Типы объектов

| Тип | Префикс | Формат ID | Описание |
|-----|---------|-----------|----------|
| Графема | G | G-0001 — G-0064 | Базовые знаки письменности |
| Лексема | L | L-000001 — ... | Слова и морфемы |
| Правило | RULE | RULE-0001 — ... | Грамматические и канонические правила |
| Искусство | ART | ART-000001 — ... | Иллюстрации, карты, арты |
| Локация | LOC | LOC-0001 — ... | Географические точки |
| Персонаж | CH | CH-0001 — ... | Исторические и современные личности |
| Событие | EV | EV-0001 — ... | Исторические события |

### 7.2. Принципы регистрации

1. Каждый уникальный объект вселенной имеет ID.
2. ID присваивается один раз и не меняется.
3. Удалённые объекты получают статус `retired`, ID не переиспользуется.
4. Ссылки на объекты всегда используют их ID.
