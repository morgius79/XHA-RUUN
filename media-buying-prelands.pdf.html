🎯 Преленды и ленды для гемблинга

Как заворачивать трафик: структура, психология, виды, техническая реализация

Креатив (Meta) → Преленд → Ленд (оффер)

1. Зачем нужен преленд

Meta Ads (креатив) — трафик
⬇ Преленд — прогрев
⬇ Ленд оффера — конверсия

Функции преленда:

  1. Прогрев пользователя — человек пришёл с картинки/видео, ему надо объяснить "что и почему"
  2. Отсев случайных — не все готовы сразу регистрироваться, преленд фильтрует
  3. Обход модерации — сам оффер может быть жёстким, преленд даёт мягкий вход
  4. Дополнительный трафик — можно получить апдейт (второй кредит) при повторе
Без прелендаС прелендом
CR в депозит1–3%5–15%
CPAВысокийНиже в 1.5–3 раза
Температура трафикаХолодный → сразу ленд → сливТёплый → заходит на ленд заинтересованным
Данные о пользователеМинимумUTM-метки, прогрев

2. Виды прелендов

Тип 1 — Статья / Новость

Выглядит как обычная статья на новостном сайте. Самый популярный для гемблинга.

Заголовок: "Секрет миллиардера из Дубая: как заработать $5000 в неделю"
Картинка: мужик в дорогом костюме с ноутбуком
Текст: 3–4 абзаца сторителлинга
Кнопка: "УЗНАТЬ ПОДРОБНОСТИ" → ленд оффера

Когда использовать: Facebook, TikTok (нативный формат)
CR переходов на ленд: 15–40%

Тип 2 — Промежуточный лендинг (витрина)

Мини-лендинг с презентацией продукта. Не сам оффер, а "обложка".

Заголовок: "Топ-10 онлайн-казино с мгновенным выводом"
Список: 3 казино с картинками
Кнопка: "ИГРАТЬ" → конкретный оффер

CR переходов: 20–50%

Тип 3 — Квиз / Опрос

Пользователь отвечает на 2–3 вопроса, в конце получает "результат" → оффер.

Вопрос 1: "Сколько вы готовы инвестировать?" → ○ до $100 ○ $100–500 ○ от $500
Вопрос 2: "Какой жанр предпочитаете?" → ○ Слоты ○ Покер ○ Ставки
→ Результат: "ВАШ ИДЕАЛЬНЫЙ ВАРИАНТ: Casino X" → ленд

CR прохождения: 30–60%

Тип 4 — ClickUnder (многослойный)

Автопереход — через N секунд или по клику в любом месте.

Когда использовать: Push-трафик, дорвеи
CR переходов: 40–70%

5. Адаптация под разные ГЕО

ГЕООсобенности преленда
Tier-1 USA, UK, CAДорогой дизайн, английский (носитель), тема инвестиций/стратегии, без откровенного "казино"
Tier-2 EU, BR, THСреднее качество, местный язык, акцент на "бесплатно/халява", яркие цвета
Tier-3 ID, PH, NG, KEПростой дизайн, местный язык, крупные кнопки, минимум текста, обещание лёгких денег
Локализация важнее дизайна. Лучше простой преленд на местном языке, чем красивый на английском в Tier-3.

Топ-3 элемента для адаптации:

  1. Имя и флаг персонажа — James (USA), João (BR), Budi (ID)
  2. Валюта — $, €, R$, ₱
  3. Стоп-слова под ГЕО — что блокирует FB в конкретной стране (узнать у партнёрки)

6. Техническая реализация

Как подключить к Keitaro

  1. Добавить оффер (из партнёрки)
  2. Создать поток (Stream): трафик с FB → преленд, преленд → ленд оффера
  3. Настроить postback между оффером и Keitaro
  4. Настроить postback между Keitaro и FB (конверсии)

Трекинг на преленде

<script>
  // 1. FB Pixel
  fbq('track', 'PageView');

  // 2. Клик по кнопке
  document.querySelector('.btn').onclick = function() {
    fbq('track', 'Lead');
  };
</script>

8. Проверенные шаблоны прелендов

1. "Программа миллионера" (сторителлинг)

Заголовок: "Секрет, который скрывают банки: как заработать $5 000 в неделю"
Структура: История обычного парня → случайно нашёл способ → скриншоты дохода → призыв
Тон: Вдохновляющий, "если смог он — сможешь ты"

2. "Новость / срочный репортаж"

Заголовок: "Шокирующие новости: эксперты обнаружили лазейку в системе"
Структура: Срочный репортаж, интервью с "экспертом", раскрытие метода
Тон: Журналистский, официальный

3. "Топ-лист / сравнение"

Заголовок: "Топ-5 платформ для заработка в 2024 году"
Структура: Список с картинками, в каждом — описание, наш — на первом месте
Тон: Обзорный, аналитический

4. One Page (ленд-без-ленда)

Когда: Если оффер не жёсткий (беттинг)
Структура: Мини-лендинг с регистрацией прямо на странице
Тон: Прямой, без лишних слоёв

9. Чек-лист хорошего преленда

  • Заголовок цепляет за 2 секунды (цифра / интрига / результат)
  • Первый абзац — крючок, перечитывание не нужно
  • Есть картинка или видео (не просто текст)
  • Кнопка контрастная, текст — действие ("Начать", "Узнать")
  • Нет стоп-слов (casino, bet, win money — удалить)
  • Работает на мобилках (мобильная вёрстка — >80% трафика)
  • Загружается за <3 секунды (сжать картинки)
  • Ссылка на оффер через Keitaro (не прямая)
  • FB Pixel / Keitaro трекинг стоят
  • Адаптирован под ГЕО (язык, персонаж, валюта)

10. Кейс: преленд для Индонезии

Оффер: Casino X, CPA $8 за депозит

Преленд:

  • Язык: Bahasa Indonesia
  • Персонаж: Budi, "водитель такси, теперь зарабатываю больше босса"
  • Валюта: IDR — "Rp 50,000,000"
  • Дизайн: простой, яркий (зелёный + жёлтый — местные цвета)
  • CTA: "COBA SEKARANG" (Попробуй сейчас)
