# Креативы для гемблинга — полный разбор

> От идеи до готового объявления: форматы, визуал, тексты, тестирование, производство

---

## 1. ЧТО ТАКОЕ ХОРОШИЙ КРЕАТИВ ДЛЯ ГЕМБЛИНГА

Хороший креатив решает 3 задачи:

1. **Привлекает внимание** — останавливает скролл за 0.5 секунды
2. **Проходит модерацию** — не содержит стоп-слов и стоп-паттернов
3. **Конвертит** — приводит к регистрации и депозиту

**Плохой креатив** = заметно, что это реклама казино → бан / низкий CTR.
**Хороший креатив** = выглядит как обычный контент, но ведёт на казино-оффер.

## 2. ФОРМАТЫ КРЕАТИВОВ ДЛЯ META

### По типу контента

| Формат | CTR (гемблинг) | Доля трафика Meta | Проходит модерацию | Сложность производства |
|--------|---------------|-------------------|-------------------|----------------------|
| **Одно изображение** | 0.5–1.5% | 40% | 🟡 Средне | 🟢 Просто |
| **Карусель** | 0.8–2.0% | 20% | 🟢 Легче | 🟡 Средне |
| **Видео 15–30 сек** | 1.0–3.0% | 30% | 🟢 Легче (AI хуже сканит) | 🔴 Сложно |
| **Reels / TikTok-style** | 1.5–4.0% | 60% растёт | 🟢 Легко (нативный формат) | 🔴 Сложно |
| **Интерактив (опрос/квиз)** | 0.5–1.0% | 5% | 🟡 Средне | 🔴 Сложно |

**Тренд 2024–2026:** Видео и Reels вытесняют статику. Meta даёт видео-форматам приоритет в показах (ниже CPM, выше охват).

## 3. ВИЗУАЛЬНЫЕ СТРАТЕГИИ — ЧТО РАБОТАЕТ

### Цветовая гамма

| Цвет | Эффект | Для гемблинга |
|------|--------|--------------|
| Чёрный + золотой | Дорого, премиально | ✅ Норм, но не перебарщивать |
| Синий + белый | Доверительно, технологично | ✅ Лучший выбор — меньше всего банят |
| Зелёный | Деньги, рост | ✅ Хорошо в сочетании с синим |
| Красный | Опасность, азарт | ❌ Триггерит модерацию |
| Фиолетовый | Премиум, тайна | ✅ Хорошо для Tier-1 |
| Жёлтый + красный | "Китайский базар" | ❌ Слишком похоже на казино-сайты |

**Рекомендуемая палитра для гемблинг-креатива:**
- Доминанта: синий (#1a5a9e) или тёмно-синий (#0d1b2a)
- Акцент: золотой (#c9a84c) или зелёный (#2d8a4e)
- Фон: светлый (#f0f2f5) или тёмный (#1a1a2e)
- Текст: белый на тёмном, тёмный на светлом

### Композиция

```
┌─────────────────────────┐
│  ВЕРХ: Заголовок (20%)   │  ← цепляет внимание
├─────────────────────────┤
│                         │
│  ЦЕНТР: Изображение     │  ← 50% площади — смысловой элемент
│         (60%)           │
│                         │
├─────────────────────────┤
│  НИЗ: CTA / кнопка      │  ← 20% — призыв к действию
└─────────────────────────┘
```

**Правило 3 секунд:** За 3 секунды человек должен понять:
1. Что это (изображение/ситуация)
2. Что ему с этого будет (выгода)
3. Куда нажимать (CTA)

### Какие изображения работают

**✅ Работает (проходит модерацию):**
- Улыбающийся человек в обычной одежде
- "До/после" (например, фото до и после с подписью)
- Технологичные изображения (ноутбук, графики, телефон с экраном)
- Абстрактные визуалы (геометрия, волны, градиенты)
- Эмодзи-визуалы (большие эмодзи + текст)

**❌ Не работает (банят / низкий CTR):**
- Откровенные деньги (пачки купюр, золото)
- Логотипы казино
- Карты, фишки, рулетка, кости
- Слишком яркие "казино-цвета"
- Скриншоты баланса/выигрышей
- Обнажёнка/полуобнажёнка

## 4. КОПИРАЙТИНГ — ТЕКСТЫ ДЛЯ META

### Структура текста объявления

**Основной текст (Primary text):**
- 1-й абзац (1–2 предложения) — крючок
- 2-й абзац — развитие/подробности
- 3-й (опционально) — социальное доказательство
- 4-й — CTA (call to action)

**Заголовок (Headline):**
- 5–10 слов
- Цифра или интрига
- Без стоп-слов

**Описание (Description):**
- Уточнение выгоды
- 1–2 предложения

**CTA кнопка:**
- "Узнать больше" / "Начать" / "Попробовать" / "Получить доступ"

### Шаблоны текстов, которые работают

**Шаблон 1 — "Секрет/лазейка"**
```
Заголовок: Он зарабатывает $X в день, пока ты читаешь это
Текст: Большинство людей даже не подозревают, что...
      Я тоже не знал, пока не попробовал...
      Теперь это мой основной доход.
Описание: $X в день. Проверено. Работает.
CTA: Узнать подробности
```

**Шаблон 2 — "Социальное доказательство"**
```
Заголовок: Уже 10 000+ человек присоединились
Текст: Средний доход участника: $X в месяц.
      Присоединяйся к сообществу успешных людей.
      Без опыта. Без вложений.
Описание: Старт за 5 минут
CTA: Начать сейчас
```

**Шаблон 3 — "Обучение/бесплатно"**
```
Заголовок: Бесплатный гайд: как заработать $X в неделю
Текст: Я собрал пошаговую инструкцию.
      Всё что нужно — интернет и 30 минут в день.
      Забирай бесплатно по ссылке.
Описание: PDF-инструкция. Без регистрации.
CTA: Скачать бесплатно
```

### Стоп-слова в текстах (полный список для гемблинга)

Ниже **ничего** не должно быть в тексте объявления:

```
casino          gambling        bet/betting
slots           poker           roulette
blackjack       jackpot         baccarat
win money       instant win     guaranteed win
no loss         รับประกัน       без риска
free money      cash prize      bonus
deposit         withdrawal      payout
lucky           777             jackpot
```

**Эвфемизмы-замены (можно использовать):**

| Нельзя | Можно |
|--------|-------|
| casino | "игровая платформа", "клуб", "партнёрская программа" |
| gambling | "развлечения", "новый способ заработка" |
| bet | "прогноз", "стратегия", "аналитика" |
| win money | "дополнительный доход", "финансовая свобода" |
| deposit | "пополнение счёта", "стартовый капитал" |
| jackpot | "главный приз", "крупный выигрыш" (осторожно) |

## 5. ВИДЕО-КРЕАТИВЫ ДЛЯ ГЕМБЛИНГА

### Почему видео работает лучше

- Meta даёт видео приоритет в алгоритме (ниже CPM на 15–30%)
- AI-модерация хуже анализирует видео, чем статику
- Видео легче маскировать (быстрая смена кадров, наложения)

### Форматы видео

**Тип 1 — Экранная запись + голос**
```
Длительность: 15–30 секунд
Содержание: запись экрана телефона, показывающая процесс
Голос: диктор объясняет "как это работает"
Музыка: фоновая, нейтральная
Финал: CTA-экран
```

**Тип 2 — Слайд-шоу (CapCut)**
```
Длительность: 10–20 секунд
Содержание: 5–8 кадров с текстом, сменяющие друг друга
Музыка: энергичная, под бит
Текст: крупный, читаемый на мобилке
Финал: лого + призыв
```

**Тип 3 — Reels / TikTok-style**
```
Длительность: 7–15 секунд
Содержание: UGC-style (снято на телефон, "живое" видео)
Монтаж: вертикальный, быстрые склейки
Субтитры: обязательны (люди смотрят без звука)
Финал: "ссылка в описании" или CTA
```

### Структура видео-креатива

```
0–2 сек:   ХУК (крючок) — зацепить внимание
           "Ты тоже хочешь зарабатывать $X в день?"
2–10 сек:  ПРОБЛЕМА → РЕШЕНИЕ
           "Раньше я сидел в офисе... а теперь..."
10–20 сек: РЕЗУЛЬТАТ (визуальный)
           Покажи график/скриншот/процесс
20–25 сек: CTA
           "Переходи по ссылке, попробуй сам"
25–30 сек: ЛОГО + БРЕНДИНГ (1–2 сек)
```

**Правило:** Первые 2 секунды решают, досмотрят ли видео. Начинай с вопроса или неожиданного кадра.

### Процесс: от идеи до запуска

1. **Сбор референсов** (30 мин) — посмотреть что льют конкуренты (Facebook Ads Library, BigSpy, AdHeart)
2. **Генерация концепта** (15 мин) — промпт в Midjourney → 4 варианта
3. **Адаптация под стоп-слова** (10 мин) — проверка текста и визуала
4. **Наложение текста + CTA** (10 мин) — Canva / Photoshop
5. **Создание 3–5 вариаций** (20 мин) — изменить цвет, текст, композицию
6. **Загрузка в Meta** (5 мин) — настроить кампанию
7. **Мониторинг результатов** — через 24 часа смотреть CTR и CR

**Время с нуля до готового креатива:** 1.5–2 часа (руками), 15–30 минут (с AI / n8n фабрикой)

## 7. ТЕСТИРОВАНИЕ КРЕАТИВОВ

### Что тестировать

| Элемент | Что менять | Какой прирост может дать |
|---------|-----------|-------------------------|
| **Изображение** | картинка полностью | +100–300% к CTR |
| **Текст объявления** | формат истории → вопрос | +20–50% к CTR |
| **Заголовок** | цифра → интрига | +10–30% к CTR |
| **CTA** | "Узнать" → "Попробовать" | +5–15% к CR |
| **Цветовая гамма** | синяя → тёмная | +10–25% к CTR |
| **Формат** | статика → видео | +30–100% к CTR |

### Протокол теста

1. Запускаешь 3–5 креативов на 1 связку (1 оффер + 1 аудитория)
2. На каждый креатив — $20–50 или до 100 кликов
3. Через 24 часа смотришь CTR и CPA
4. **Оставляешь** топ-1 креатив (лучший CTR / CPA)
5. **Отключаешь** всё остальное
6. На основе топ-1 делаешь 3–5 вариаций (изменить цвет, текст, CTA)
7. Повторяешь

**Вариант теста (для экономии бюджета):**
- Запускаешь 1 креатив на 3 разные аудитории
- Победившую аудиторию → тестируешь 3 креатива
- Победивший креатив → масштабируешь

### Критерии оценки

| Метрика | Отлично | Норм | Плохо | Действие |
|---------|---------|------|-------|----------|
| CTR | >1.5% | 0.5–1.5% | <0.3% | Плохой → KILL |
| CPA | <50% выплаты | 50–80% | >100% | Высокий → KILL |
| CPC | <$0.50 (T3) | $0.5–1.5 | >$2 | Высокий → сменить таргет |
| Frequency за 3 дня | <3 | 3–5 | >5 | Высокий → обновить креатив |

## 8. ЖИЗНЕННЫЙ ЦИКЛ КРЕАТИВА

```
День 1-2:   ТЕСТ — бюджет $20–50, смотреть CTR
День 3-5:   ПОБЕДА — если CTR >0.8%, увеличиваю бюджет
День 6-10:  МАСШТАБ — полный бюджет, мониторинг frequency
День 10+:   СМЕРТЬ — CTR упал, frequency >5 → выключать
            НОВАЯ ВОЛНА — запускать свежий креатив
```

**Признаки, что креатив умер:**
- CTR упал ниже 0.3%
- Frequency >5
- CPA вырос в 2+ раза за 2 дня
- CPC растёт, а CR падает

**Что делать:** Выключить. Не пытаться реанимировать. Делать новый.

## 9. ПРОДВИНУТЫЕ ТЕХНИКИ

### Техника 1 — "Маскировка под блогера"
Креатив выглядит как пост блогера: фото человека, текст "от первого лица", без брендинга. Пользователь думает что это органика, Meta модерирует как обычный контент.

### Техника 2 — "Комикс / комик-стрип"
Серия из 3–4 картинок в формате комикса. Каждая панель — шаг истории. Meta не распознаёт как рекламу казино, если нет стоп-паттернов.

### Техника 3 — "Вопрос без ответа"
Креатив задаёт вопрос, который провоцирует любопытство. Пользователь кликает чтобы узнать ответ. Высокий CTR, низкая цена.

Примеры:
- "Почему банкиры увольняются в 30 лет?"
- "Что скрывают владельцы казино?"
- "О чём молчат финансовые консультанты?"

### Техника 4 — "Анимация из статики"
Берёшь статичное изображение, добавляешь микро-анимацию (движение фона, мерцание текста, пульсирующая кнопка). Meta считает это видео — проходит модерацию легче.

Инструменты: CapCut (анимация по ключевым кадрам) / Canva (анимированные шаблоны)

## 11. ЧЕК-ЛИСТ ПЕРЕД ЗАПУСКОМ КРЕАТИВА

- [ ] Визуал не содержит стоп-паттернов (деньги, карты, фишки, лого казино)
- [ ] Цветовая гамма безопасная (нет кислотных/казино-цветов)
- [ ] Текст объявления не содержит стоп-слов
- [ ] Заголовок содержит цифру или интригу
- [ ] CTA — действие ("Узнать", "Начать", "Попробовать")
- [ ] Креатив выглядит как обычный контент (не как реклама казино)
- [ ] Первые 2 секунды видео — крючок
- [ ] Видео имеет субтитры (люди смотрят без звука)
- [ ] Формат соответствует правилам размещения (1080×1080, 1080×1920)
- [ ] Создано 3–5 вариаций для теста
- [ ] Бюджет на тест — $20–50 на креатив
- [ ] Преленд/оффер прикручен через Keitaro
- [ ] FB Pixel стоит (или CAPI Server)
