# Преленды и ленды для гемблинга — полный разбор

> Как заворачивать трафик: структура, психология, виды, техническая реализация

---

## 1. ЗАЧЕМ НУЖЕН ПРЕЛЕНД

**Схема:** `Креатив (Meta) → Преленд → Ленд (оффер)`

### Функции преленда:

1. **Прогрев пользователя** — человек пришёл с картинки/видео, ему надо объяснить "что и почему"
2. **Отсев вёрджинов (чистых)** — не все готовы сразу регистрироваться в казино, преленд отсеивает случайных
3. **Обход модерации** — сам оффер может быть жёстким (казино), преленд даёт мягкий вход
4. **Дополнительный трафик** — можно получить апдейт (второй кредит) если пользователь не конвертился на первом ленде

### Почему это важно:

| Без преленда | С прелендом |
|-------------|-------------|
| CR в депозит: 1–3% | CR в депозит: 5–15% |
| Высокий CPA | CPA ниже в 1.5–3 раза |
| Холодный трафик идёт сразу на ленд → слив | Тёплый трафик заходит на ленд уже заинтересованным |
| Меньше данных о пользователе | Можно передать UTM-метки в трекер |

## 2. ВИДЫ ПРЕЛЕНДОВ

### Тип 1 — Статья / Новость (самый популярный для гемблинга)

Выглядит как обычная статья на новостном сайте. Пользователь читает "новость", а в конце — переход к офферу.

```
Заголовок: "Секрет миллиардера из Дубая: как заработать $5000 в неделю"
Картинка: мужик в дорогом костюме с ноутбуком
Текст: "Мало кто знает, но существует способ..."
        (3-4 абзаца сторителлинга)
Кнопка: "УЗНАТЬ ПОДРОБНОСТИ" → ленд оффера
```

**Когда использовать:** Facebook, TikTok (нативный формат)
**CR:** 15–40% переходов на ленд

### Тип 2 — Промежуточный лендинг (витрина)

Мини-лендинг с презентацией продукта. Не сам оффер, а "обложка" для него.

```
Заголовок: "Топ-10 онлайн-казино с мгновенным выводом"
Список: 3 казино с картинками
Кнопка: "ИГРАТЬ" → конкретный оффер
```

**Когда использовать:** Когда у оффера нет своего крутого ленда
**CR:** 20–50% переходов на ленд

### Тип 3 — Квиз / Опрос

Пользователь отвечает на 2–3 вопроса, в конце получает "результат" → оффер.

```
Вопрос 1: "Сколько вы готовы инвестировать в месяц?"
   ○ до $100  ○ $100–500  ○ от $500
Вопрос 2: "Какой жанр предпочитаете?"
   ○ Слоты  ○ Покер  ○ Ставки на спорт

→ "ВАШ ИДЕАЛЬНЫЙ ВАРИАНТ: CASINO X" → ленд
```

**Когда использовать:** Чтобы собрать данные о пользователе или повысить вовлечение
**CR:** 30–60% проходят весь квиз

### Тип 4 — ClickUnder (многослойный)

Преленд с автопереходом — пользователь видит контент, а через N секунд (или по клику в любом месте) его перебрасывает на ленд. Либо всплывающее окно (popunder).

```
Картинка + текст на фоне
Фон: затемнение
Через 5 секунд → кнопка "ПРОДОЛЖИТЬ"
Или: клик в любом месте → редирект на ленд
```

**Когда использовать:** Push-трафик, дорвеи
**CR:** 40–70% переходов

## 3. ПСИХОЛОГИЯ НАТИВНОГО ПРЕЛЕНДА

Эффективный преленд строится по формуле AIDA:

| Этап | Что происходит | Пример |
|------|---------------|--------|
| **A**ttention (внимание) | Зацепить заголовком | "Он бросил работу и зарабатывает $2000 в день" |
| **I**nterest (интерес) | Вовлечь историей | "Всё началось с эксперимента..." |
| **D**esire (желание) | Показать выгоду | "За первый месяц — $5000 чистыми" |
| **A**ction (действие) | Призвать к переходу | "Попробуй прямо сейчас →" |

### Адаптация для гемблинг-вертикали:

**Триггеры, которые работают:**
- **FOMO (Fear Of Missing Out)** — "пока вы читаете, другие уже забирают выигрыш"
- **Социальное доказательство** — "уже 10 000+ пользователей, 4.8★ рейтинг"
- **Лёгкость старта** — "регистрация за 30 секунд, депозит от $10"
- **Персонаж-авторитет** — "финансовый аналитик раскрыл схему"

**Стоп-слова (нельзя в преленде при платном трафике):**
- Слова, которые блокирует Meta: casino, gambling, slots, bet, win money, jackpot
- Использовать эвфемизмы: "игровая платформа", "развлечения", "клуб", "партнёрская программа", "инвестиции"

## 4. СТРУКТУРА ПРЕЛЕНДА НА HTML

Вот базовая структура преленда-статьи для гемблинга:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>История успеха</title>
  <style>
    body { font-family: Arial, sans-serif; max-width: 680px; margin: 0 auto; padding: 20px; }
    h1 { font-size: 24px; line-height: 1.3; }
    .meta { color: #666; font-size: 13px; margin: 10px 0 20px; }
    .content { font-size: 16px; line-height: 1.7; color: #333; }
    .content p { margin: 14px 0; }
    .content img { max-width: 100%; border-radius: 8px; margin: 10px 0; }
    .btn {
      display: block; max-width: 320px; margin: 24px auto;
      padding: 14px 28px; background: #1a8917; color: white;
      text-align: center; text-decoration: none; border-radius: 6px;
      font-size: 18px; font-weight: bold;
    }
  </style>
</head>
<body>
  <h1>Он бросил работу и зарабатывает $2000 в день. Реальная история</h1>
  <div class="meta">📅 15 июля 2024 • 📍 Автор: Редакция</div>
  <div class="content">
    <img src="hero.jpg" alt="фото">
    <p>В прошлом месяце Алексей из Краснодара уволился с работы. Не потому что выиграл в лотерею, а потому что нашёл способ зарабатывать больше, чем все его знакомые вместе взятые.</p>
    <p>...</p>
    <p>Сейчас Алексей зарабатывает от $2000 в день. Проверено — выплаты приходят вовремя.</p>
    <a class="btn" href="https://trak.link/?offer=XXX">УЗНАТЬ ПОДРОБНОСТИ</a>
  </div>
  <script>
    // трекинг: отправить сигнал в Keitaro
    fetch('/visit', { /* ... */ });
  </script>
</body>
</html>
```

### Важные элементы:

| Элемент | Зачем | Реализация |
|---------|------|-----------|
| **Заголовок-кликбейт** | Чтобы открыли | 8–12 слов, цифра, интрига |
| **Мета-дата** | Создаёт ощущение реальности | "15 июля 2024 • Автор: Редакция" |
| **Фото героя** | Визуальное доверие | Стоковое фото + лёгкая обработка |
| **Абзацы 3–5** | Сторителлинг | Проблема → решение → результат |
| **Кнопка CTA** | Переход на ленд | Контрастный цвет, "УЗНАТЬ" / "ПОЛУЧИТЬ" |
| **Трекинг** | Статистика | Пиксель FB / Keitaro pixel |

## 5. АДАПТАЦИЯ ПОД РАЗНЫЕ ГЕО

То, что работает на USA, провалится на Индонезии. Преленд нужно адаптировать под ГЕО.

| ГЕО | Особенности |
|-----|------------|
| **Tier-1 (USA, UK, CA)** | Дорогой дизайн, английский (носитель), акцент на "стратегию" а не "казино", темы инвестиций |
| **Tier-2 (EU, BR, TH)** | Среднее качество, местный язык, акцент на "халява/бесплатно", яркие цвета |
| **Tier-3 (ID, PH, NG, KE)** | Простой дизайн, местный язык, крупные кнопки, минимум текста, обещание лёгких денег |

**Локализация важнее дизайна.** Лучше простой преленд на местном языке, чем красивый на английском в Tier-3.

### Топ-3 элемента для адаптации:
1. **Имя и флаг персонажа** — не Алексей, а James (USA), João (BR), Budi (ID)
2. **Валюта** — $, €, R$, ₱
3. **Стоп-слова** — что блокирует FB в этом ГЕО (список от партнёрки)

## 6. ТЕХНИЧЕСКАЯ РЕАЛИЗАЦИЯ

### Как подключить к Keitaro

```
В Keitaro:
1. Добавить оффер (из партнёрки)
2. Создать поток (Stream):
   - Трафик с FB → преленд (указать URL преленда)
   - Если преленд сработал → ленд оффера
   - Если пользователь не конвертился через N дней → другой преленд / другой ленд
3. Настроить postback между оффером и Keitaro
4. Настроить postback между Keitaro и FB (конверсии)
```

**Трекинг на преленде:**
```html
<script>
  // 1. Уведомить Keitaro о визите
  // (Keitaro подставляет свой пиксель автоматически)
  
  // 2. FB Pixel (если не хочешь светить прямо — через CAPI Server)
  fbq('track', 'PageView');
  
  // 3. Событие клика по кнопке
  document.querySelector('.btn').onclick = function() {
    fbq('track', 'Lead');
    // Keitaro сам проксирует переход, ничего дополнительно не нужно
  };
</script>
```

### Хостинг прелендов

| Способ | Плюсы | Минусы |
|--------|-------|--------|
| **Keitaro (встроенный)** | Просто, всё в одном | Дорого на объёмах |
| **Аренда VPS** + свой домен | Дешево, полный контроль | Настраивать самому |
| **GitHub Pages / Cloudflare Pages** | Бесплатно | Только статика |
| **Купить готовый хост у партнёрки** | 0 хлопот | Нет контроля |

**Для старта:** сначала используй готовый преленд от партнёрки. Когда поймёшь что работает — сделаешь свои.

## 7. КАК ТЕСТИРОВАТЬ ПРЕЛЕНДЫ

### Метрики преленда:

| Метрика | Норма | Что показывает |
|---------|-------|---------------|
| **Переход на ленд** (Click-to-Offer) | 15–40% от зашедших на преленд | Насколько зашёл контент |
| **Время на странице** | >15 секунд | Зацепил ли контент |
| **Bounce Rate** | <60% | Люди уходят или читают |
| **CR преленда** (депозиты ÷ заходы на преленд) | >3% | Общая эффективность преленда |

### Тест: 2 варианта → 50 кликов → вывод

Не парься с A/B-тестами. Просто сделай 2 разных преленда на 1 оффер, залей на 50 кликов каждый. Чей CR выше — тот и топ.

## 8. ПОПУЛЯРНЫЕ ШАБЛОНЫ ПРЕЛЕНДОВ

Вот 4 проверенных шаблона для гемблинга:

### 1. "Программа миллионера" (сторителлинг)
```
Заголовок: "Секрет, который скрывают банки: как заработать $LAУХ в неделю"
Структура: История обычного парня → случайно нашёл способ → скриншоты дохода → призыв
Тон: Вдохновляющий, "если смог он — сможешь ты"
```

### 2. "Новость / срочный репортаж"
```
Заголовок: "Шокирующие новости: эксперты обнаружили лазейку в системе"
Структура: Срочный репортаж, интервью с "экспертом", раскрытие метода
Тон: Журналистский, официальный
```

### 3. "Топ-лист / сравнение"
```
Заголовок: "Топ-5 платформ для заработка в 2024 году"
Структура: Список с картинками, в каждом — описание, наш — на первом месте
Тон: Обзорный, аналитический
```

### 4. "Ленд-без-ленда" (One Page)
```
Преленд и оффер — одно и то же.
Структура: Мини-лендинг с регистрацией прямо на нём
Тон: Прямой, без лишних слоёв
Когда: Если оффер не жёсткий (например, беттинг)
```

## 9. ЧЕК-ЛИСТ ХОРОШЕГО ПРЕЛЕНДА

- [ ] Заголовок цепляет за 2 секунды (есть цифра/интрига/результат)
- [ ] Первый абзац — крючок, перечитывание не нужно
- [ ] Картинка/видео есть (не просто текст)
- [ ] Кнопка контрастная, текст на кнопке — действие ("Начать", "Узнать")
- [ ] Нет стоп-слов (casino, bet, win money — удалить)
- [ ] Работает на мобилках (мобильная вёрстка >80% трафика в гемблинге)
- [ ] Загружается за <3 секунды (сжать картинки)
- [ ] Ссылка на оффер замаскирована (через Keitaro, не прямая)
- [ ] Treking-пиксели стоят (FB Pixel / Keitaro)
- [ ] Адаптирован под ГЕО (язык, персонаж, валюта)

## 10. КЕЙС: ПРЕЛЕНД ДЛЯ ИНДОНЕЗИИ

Оффер: Casino X, CPA $8 за депозит

**Преленд:**
- Язык: Bahasa Indonesia
- Персонаж: Budi, "я водитель такси, теперь зарабатываю больше босса"
- Валюта: IDR (рупии) — "Rp 50,000,000"
- Дизайн: простой, яркий (зелёный и жёлтый — местные цвета)
- CTA: "COBA SEKARANG" (Попробуй сейчас)

**Результат:**
- Переход на оффер: 38%
- CR преленда (деп): 7.2%
- CPA: $4.6 (против $8 выплаты → +73% ROI)

**Вывод:** Хороший преленд для Tier-3 может давать CPA в 2 раза ниже выплаты.
