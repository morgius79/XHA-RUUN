Volume IV: Language — Chapter 7: Sharuu Writing System

"Sharuu is not an alphabet. Sharuu is a world. Each character is not merely a sound, but a concept. 124 syllabic signs and 48 logograms — keys to the universe." — From "History of Writing," Academy

Related sections: → see [Ch.06, "Language"], → see [Ch.03, "Writing System"] Length: 30 pages

7.1 Script Type

Mixed: syllabary (124 signs) + logograms (48 signs).

Component Count Function
Syllabic signs (V, CV, CCV) 124 Phonetic writing
Logograms 48 Semantic roots
Numerals (senary) 6 0–5
Punctuation 8 Separation

7.2 Direction

Right to left, top to bottom. Paragraphs arranged in columns.

7.3 Graphemes

Modular grid 16×16. 64 base graphemes (G-0001–G-0064), combining into syllabic signs.
