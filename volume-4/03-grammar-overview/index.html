Volume IV: Language — Chapter 3: Grammar: Overview

"The Xha'Ruun language is agglutinative. Each morpheme is a building block from which utterances are constructed. 14 cases, 4 tenses, 3 numbers, 4 types of evidentiality — precision that tolerates no ambiguity." — Academic of Grammar, Tor-Ven

Related sections: → see [Ch.06, "Language"], → see [Volume IV, Ch.1–2] Length: 25 pages

3.1 Typological Characteristics

Parameter Value
Type Agglutinative, suffixal
Word order SOV (subject-object-verb)
Cases 14
Numbers 3 (singular, dual, plural)
Gender None
Evidentiality Mandatory (4 types)
Tense 3 + 1 (past, present, future + gnomic)

3.2 Word Structure

Stem-(root) + number_suffix + case_suffix + (evidentiality)

Example: xha'ruun-ak-sha-ven → "people-PL-ERG-DIR" = "the peoples (I see them)"
