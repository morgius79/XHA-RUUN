Volume IX: Dictionary — Chapter 12: Xha'Ruun Name Guide

"A name is not a label. A name is destiny. To give a name is to determine a path." — From the "Book of Names," Naming Ceremony

12.1 Name Structure

Xha'Ruun names are two-part: [root-1]-[root-2]. Stress falls on the second part.

Examples:

  • Khar-Vexar — "Light-Giver"
  • Ven-Khal — "Truth-Life"
  • Shan-En — "Flame-Being"

12.2 Gender Endings

Gender Ending Example
Male (♂) -en, -un Verksen, Torun
Female (♀) -a, -ae Thexa, Rhuae
Neuter (⚲) -in, -el Sharin, Vexel

End of Chapter 12, Volume IX.
