## 1. ПОЧЕМУ БАНЯТ — СИСТЕМА ДЕТЕКЦИИ FACEBOOK

Meta использует многоуровневую систему проверки. Понимание каждого уровня = понимание, как его обходить.

### Уровень 1 — Автоматика (AI-модерация)

| Что проверяет | Как работает | Как обходить |
|--------------|-------------|-------------|
| **Креативы** (картинка/видео + текст) | ML-модель сканирует изображение и текст на стоп-слова и стоп-паттерны | Эвфемизмы, размытие, реверсивная картинка, текст на картинке |
| **Посадочная страница** | Проверяет ссылку в объявлении — сканирует контент на landing page | Преленд, закрытый от сканирования |
| **Платёжные данные** | Сверяет страну карты, IP, аккаунта | Совпадение всех трёх |
| **Поведение аккаунта** | Частота действий, резкие изменения | Прогрев, плавный старт |

### Уровень 2 — Полуавтоматика

Если AI что-то заподозрил (confidence score 60–90%), объявление отправляется на ручную проверку.

**Триггеры для полуавтоматики:**
- Жалобы пользователей (самый частый)
- Резкий рост бюджета (в 5+ раз за день)
- Креатив с пограничным контентом (AI не уверен)
- Аккаунт без истории

### Уровень 3 — Ручная модерация

Реальный сотрудник Meta смотрит креатив и ленд. Если видит нарушение — бан.

**Что видит модератор:**
- Ваше объявление (креатив + текст)
- Посадочную страницу (если доступна)
- Историю аккаунта (сколько живёт, были ли нарушения)
- Жалобы пользователей

## 3. УРОВЕНЬ 1 — ЗАЩИТА КРЕАТИВА

### Что триггерит бан в креативе

**Текст объявления:**
- Прямые: casino, gambling, bet, slots, poker, roulette, blackjack, jackpot, win money, cash prize
- Косвенные: guarantee, no loss, 100% win, free money, instant withdrawal
- Призывы: click here, buy now (без контекста)

**Картинка/видео:**
- Логотипы казино (PokerStars, 1xBet, etc.)
- Деньги, пачки купюр, золото
- Фишки, карты, рулетка, кости, игровые автоматы
- Скриншоты выигрышей с конкретными суммами
- Слишком яркие/глиттерные цвета (как у казино)

**Домен:**
- Слова casino, bet, win, slot в URL
- Домен, зарегистрированный вчера
- IP, который уже светился в казино-рекламе

### Правила безопасного креатива

| Компонент | Safe | Danger |
|-----------|------|--------|
| **Цвета** | Приглушённые, пастельные, "технологичные" (синий/серый/чёрный) | Кислотные, золото/красный (цвета казино) |
| **Люди** | Обычные люди в обычной одежде | Люди в дорогих костюмах с деньгами |
| **Атмосфера** | Современная, технологичная | Игорный дом, казино-зал |
| **Текст на картинке** | Названия бренда нет, слоганы общие | "Бесплатные вращения", "Джекпот" |
| **Лого** | Свой бренд (который ничего не значит) | Лого известного казино |

### Техники маскировки

**1. Размытие / шум:** наложить на картинку лёгкий шум (2–5%), который AI не может распарсить, но человек видит норм
**2. Реверсивная картинка:** отзеркалить, изменить цветовую гамму, обрезать — чтобы не совпадал хеш с забаненными креативами
**3. Текст поверх текста:** AI видит текст на картинке — пиши его с опечатками или частично перекрывай элементами
**4. Видео вместо картинки:** Видео модерируется мягче — AI сложнее анализировать движущиеся кадры
**5. Смена формата каждые 2–3 дня:** Не давать AI накопить статистику на твой креатив

## 4. УРОВЕНЬ 2 — ЗАЩИТА ПРЕЛЕНДА

### Как модерация проверяет ленд

Когда ты загружаешь объявление, бот Meta делает запрос на URL, указанный в объявлении. Если он видит контент казино — бан.

### Методы обхода

**Метод 1 — Geo-Targeting преленда:**
```nginx
if ($http_user_agent ~* (facebook|meta)) {
    return 301 /clean.html;  # для бота — чистая страница
}
# для реального пользователя — преленд
```

Показывать ботам Meta пустую или乾淨 (чистую) страницу, а реальным пользователям — преленд.

**Метод 2 — IP-based блокировка:**
Заблокировать доступ для IP-диапазонов Meta, Facebook, Instagram. Преленд открывается только с "обычных" IP.

Список IP-диапазонов Meta: https://developers.facebook.com/docs/graph-api/ip-addresses/

**Метод 3 — JS-редирект (менее надёжный, но проще):**
```javascript
// Если пользователь пришёл не с рекламного клика — показать чистую страницу
if (!document.referrer.includes('facebook.com') && !document.referrer.includes('m.facebook.com')) {
    window.location.href = '/clean.html';
}
```

**Метод 4 — Только преленд, без оффера напрямую:**
Трафик идёт: Meta → Преленд → Keitaro → Оффер (ленд партнёрки).
Meta видит только преленд, никогда — оффер.

### Что должно быть на преленде для модерации

Если модератор всё-таки зашёл на преленд, он должен увидеть:

- [ ] Нет стоп-слов (casino, bet, slots, etc.)
- [ ] Нет упоминаний азартных игр
- [ ] Нет кнопок "Играть", "Ставки"
- [ ] Выглядит как обычный блог/статья/новость
- [ ] Есть политика конфиденциальности (юридически важно!)
- [ ] Есть контактные данные (или видимость)

## 5. УРОВЕНЬ 3 — АККАУНТЫ И ПРОКСИ

### Аккаунты

**Никогда не используй свой личный аккаунт!**

**Вариант А — Купить фармленный (рекомендуется для старта):**

| Тип | Живучесть | Цена | Где брать |
|-----|-----------|------|-----------|
| Фармленный 30+ дней | 2–6 недель | $20–50 | CheckTrust, Accfarm, UserTeam |
| Фармленный 60+ дней | 1–3 месяца | $50–150 | Те же, но с историей |
| С RMP (Reels) | дольше | +$20 | У тех же продавцов |
| Индийские | дёшево, живут меньше | $5–10 | Магазины акков |

**Вариант Б — Фармить самому:**
- Заводишь акк с чистой симки
- 7–14 дней заполняешь профиль
- Добавляешь 30+ друзей (рассылка приглашений)
- Постишь каждый день (хотя бы репосты)
- Смотришь Reels, ставишь лайки
- Покупаешь через него (мелкие платежи)
- Через 30+ дней — можно лить

**Сигналы, которые ведут к банам аккаунтов:**
- Вход с необычного местоположения (по сравнению с историей)
- Использование публичного/VPN-IP (который уже банился)
- Быстрое добавление большого количества друзей
- Массовая рассылка сообщений
- Резкая смена имени или фото профиля
- Использование одного IP для нескольких аккаунтов

### Прокси

| Тип | Оценка | Для чего |
|-----|--------|----------|
| **IPv4 резидентные** (лучшие) | 🏆 | Заход в аккаунты, работа с BM |
| **IPv4 дата-центр** (жирные) | ⭐ | Для AI инструментов, не для акков |
| **IPv6** (дешёвые) | ⚠️ | Meta плохо их понимает, частые баны |
| **Mobile proxy** | 👍 | Хорошо, но дорого и медленно |

**Правила:**
- 1 аккаунт = 1 выделенный прокси (только твой, никто больше)
- IP должен совпадать со страной аккаунта
- Не использовать прокси, который уже банили в Meta (проверить через https://whatismyipaddress.com/blacklist-check)
- Прокси должен быть жирным (не забаненным на других сайтах)
- Менять прокси каждые 1–2 месяца (аккуратно, не дёргая акк)

### Антидетект-браузеры

Обычный Chrome сдаёт fingerprint (отпечаток браузера): разрешение экрана, шрифты, Canvas fingerprint, WebGL, User-Agent, часовой пояс, язык системы.

**Зачем антидетект:** Создать уникальный fingerprint для каждого аккаунта, чтобы Meta не поняла, что 5 акков ведёт один человек.

**Популярные:**
- **Indigo** — российский, адекватные цены
- **Dolphin{anty}** — до 10 профилей бесплатно
- **Multilogin** — топ, но дорогой
- **GoLogin** — дешевле Multilogin

**Минимальные настройки fingerprint:**
- User-Agent: под реальное устройство (Windows/Mac)
- Разрешение: 1920×1080 или 1366×768
- Часовой пояс: страны аккаунта
- Язык: страны аккаунта
- WebGL: подменить под реальную видеокарту
- Шрифты: подгрузить системные шрифты страны аккаунта

## 6. УРОВЕНЬ 4 — ИНФРАСТРУКТУРА

### Business Manager

**BM — это твой щит.** Когда банят рекламный аккаунт, BM (Business Manager) должен оставаться жив. Если банят BM — ты теряешь всё сразу.

**Правила BM:**
- Не делай всё в одном BM. Распределяй риски.
- На каждый BM: 2+ админа (если можно — пригласить резидента страны)
- BM должен иметь историю: 3+ месяца, были рекламные кампании
- Подтверждённый бизнес (верификация) — продлевает жизнь BM в разы
- К BM привязана страница (с историей, постами)

### Рекламные аккаунты

| Стратегия | Риск | Бюджет |
|-----------|------|--------|
| 1 BM = 5 РК | Средний (потеряешь 5 за раз) | Низкий |
| 1 BM = 1 РК, но много BM | Низкий | Высокий |
| Агентские аккаунты через партнёрку | Низкий | 5–10% комиссии |

**Рекомендация для старта:** 1 BM → 2–3 рекламных аккаунта → разные креативы на разных акках.

### Платёжные карты

**Проблема:** Карта должна быть выпущена в той же стране, что и аккаунт. Карта РФ не работает с USA-акком.

**Решения:**
- **Capitalist** — популярный вариант для СНГ-баеров, даёт виртуальные карты
- **VCC (Virtual Credit Card)** — купить у поставщиков (карты разных стран)
- **Банковские карты местных** — если есть партнёр в стране
- **AdvCash / Payoneer** — ограниченно, но работают

**Правило:** Страна карты = Страна аккаунта = IP прокси.

### Страницы Facebook

К BM нужна привязанная страница. Не создавай новую — купи готовую с историей (хотя бы 1+ месяц, есть посты, подписчики).

## 7. РЕЖИМЫ ЗАПУСКА — ОТ МЯГКОГО К ЖЁСТКОМУ

### Режим A (безопасный, но дорогой)
- Фармленный акк (60+ дней)
- Жирный резидентный прокси
- Антидетект-браузер
- Преленд с гео-таргетингом (Meta видит чистое)
- 1 акк → 1 прокси → 1–2 кампании в день
- Бюджет: по $20–50/день на акк

### Режим B (рабочий, баланс)
- Фармленный акк (30+ дней)
- Дата-центр IPv4 (чистый, проверенный)
- Антидетект-браузер
- Преленд без жёстких стоп-слов
- 1 акк → 3–5 кампаний
- Бюджет: по $50–100/день на акк

### Режим C (агрессивный, высокий риск)
- Свежие акки (7–14 дней)
- Дата-центр IPv4 (обычный)
- Обычный браузер (чистый профиль Chrome)
- Преленд с JS-редиректом
- Бюджет: $100–300/день на акк — надо успевать отливать до бана

## 8. ЧТО ДЕЛАТЬ, ЕСЛИ ЗАБАНИЛИ

### Этап 1 — Диагностика

Определи уровень бана:

| Тип бана | Что случилось | Можно ли вернуть |
|----------|--------------|------------------|
| **Креатив отклонён** | Не прошёл модерацию | Да — подать аппеляцию |
| **Рекламный аккаунт забанен** | Нарушение политики рекламы | 50/50 — аппеляция |
| **Личный аккаунт забанен** | Серьёзное нарушение | Сложно |
| **BM забанен** | Системное нарушение | Очень сложно |
| **Страница забанена** | Контент страницы | Можно попробовать |

### Этап 2 — Аппеляция

**Шансы на успех:**
- Первый бан: 60–70% (подавай аппеляцию)
- Второй бан: 20–30%
- Третий+: <5%

**Шаблон аппеляции (для рекламного аккаунта):**
```
Hello,

I believe my ad account was disabled by mistake. I have reviewed 
the Advertising Policies and believe my ads comply.

I run ads for [мой бизнес/блог/—], and I always make sure my 
ads follow the guidelines. I didn't intend to violate any policies.

Please review my case. I care about advertising on your platform 
and will be more careful in the future.

Best regards,
[Имя]
```

**Важно:** Никогда не пиши в аппеляции, что ты рекламируешь казино. Пиши общими фразами.

### Этап 3 — Если аккаунт не вернули

- Сменить прокси (старый мог попасть в чёрный список)
- Сменить антидетект-профиль (fingerprint мог быть сохранён)
- Купить новый аккаунт (старый — в утиль)
- Проверить креатив и преленд на стоп-слова
- Запускаться через месяц на новом акке, а не сразу

## 9. ЧЕК-ЛИСТ ПЕРЕД ЗАПУСКОМ КАМПАНИИ

**Перед каждым запуском проверяй:**

- [ ] Аккаунт фармленный (30+ дней)
- [ ] Прокси жирный, страна = стране аккаунта
- [ ] Антидетект настроен (fingerprint под страну)
- [ ] Креатив проверен на стоп-слова (нет casino, bet, win)
- [ ] Текст объявления чистый (без стоп-слов)
- [ ] Преленд не показывается ботам Meta (geo-targeting / JS)
- [ ] Преленд не содержит стоп-слов
- [ ] Ссылка идёт через Keitaro (не прямая)
- [ ] Домен не засвечен в казино раньше
- [ ] Карта = страна аккаунта
- [ ] Бюджет на старте — минимальный (прогрев)

**Еженедельно:**
- [ ] Менять креативы (хотя бы 1 раз в 3–4 дня)
- [ ] Проверять живучесть прокси
- [ ] Закупать новые аккаунты (чтобы был запас)
- [ ] Смотреть, какие креативы живут дольше

## 10. КОЛИЧЕСТВЕННАЯ МОДЕЛЬ — СКОЛЬКО АККАУНТОВ НУЖНО

```
Исходные данные:
- Средняя жизнь аккаунта: 14 дней
- Бюджет на акк/день: $30
- Время на фарминг нового: 30 дней (если сам)

Расход аккаунтов:
- 1 акк живёт 14 дней
- В месяц: 30 / 14 ≈ 2 акка в месяц
- На старте нужно 2-3 акка (чтобы лить, пока один прогревается)

Для стабильной работы (на одного человека):
- В обороте: 3 акка (1 льёт, 1 прогревается, 1 запас)
- Расход: 6–8 акков в месяц
- Бюджет на акки: $30–150/мес (в зависимости от качества)
```

## 11. ОСНОВНЫЕ ПРАВИЛА (ЗАПОМНИТЬ)

| Правило | Почему |
|---------|--------|
| **Никогда** не используй личный акк | Потеряешь всё — друзей, страницы, сообщения |
| **1 акк = 1 прокси** | Meta отслеживает связь по IP |
| **Не свети оффер напрямую** | Всегда через преленд + Keitaro |
| **Не лей $300 в день на новом акке** | Прогревай — $20 → $50 → $100 |
| **Меняй креативы каждые 2–3 дня** | Не давай AI накопить статистику |
| **Не делай всё в одном BM** | BM банят реже, но больнее |
| **Аппеляция — это бесплатно** | Подавай всегда, даже если уверен что не вернут |
| **Страна карты = страна акка = IP** | Любое несовпадение — триггер |
| **Фармленный акк 30+ дней живёт дольше свежего** | В 3–5 раз |
| **NEVER fall in love with an account** | Акканты расходники. Купил — используешь — потерял — купил следующий |
